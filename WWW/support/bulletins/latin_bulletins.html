Boletines

En esta página encontrará boletines informativos, técnicos y de software de los productos AST. A continuación encontrará una descripción de la información contenida en esta sección:


Boletines Técnicos de AST (TBs): Estos documentos contienen las últimas soluciones a problemas técnicos. Cada Boletín Técnico tiene una descripción del problema, la causa y la solución.

Boletines de Software de AST (SBs): Estos documentos contienen información al día acerca del software. Cada Boletín de Software describe el problema (si lo hay), la solución (si aplicable) y cualquier mejora al software existente.

Boletines Informativos de AST (IBs): Estos documentos contienen información técnica usualmente no relacionada con hardware o software de AST. Esta sección esta incluida para su información.

Si conoce el nombre o la clave para el documento que necesita puede indicarlo en la sección "<--Search For" con el nombre del documento y su clave y oprimir el botón "<--Search For".
