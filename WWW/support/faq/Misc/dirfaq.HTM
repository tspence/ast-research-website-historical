What is DirectX technology?

Microsoft created DirectX drivers to provide resources for real-time applications within Windows® 95. DirectX drivers are not applications, like typical drivers, but sets of programming interfaces that allow the application to access the system's multimedia hardware. Because of the popularity of Windows 95, most game programmers now incorporate DirectX technology into the code of a game. Additionally, many programmers that incorporate DirectX features within their games choose to use DirectX Advanced Programming Interfaces (APIs). The software author can program for standard DirectX capabilities, rather than develop the software around the multitude of different drivers. DirectX provides a standard set of APIs that a programmer can use. There are several different sets of the DirectX programming APIs including: DirectDraw, DirectSound, Direct3D, DirectPlay, and DirectInput.

What is DirectInput?

DirectInput provides low-latency support for analog and digital joysticks. Within the majority of AST systems, the audio chipset controls the joystick. When you update the drivers for the audio chipset you will install the appropriate DirectInput updates for the joystick controller. Some games also rely on the mouse and/or keyboard as the DirectInput device.

What is Direct3D?

Direct3D is a real-time, interactive, programming interface. This interface relies on the other DirectX APIs to provide interactive 3-D technology. New Direct3D applications will provide enhanced 3-D features within games, the Internet, as well as interactive media.

After I played a game that uses DirectX technology, I had a problem with my video (audio); what should I do?

Often, games and other applications that use DirectX programming will alter some of the system settings when you return to Windows. In some scenarios, the system will not boot; some of the Control Panel settings may change, or the system will “freeze.” Generally, you can resolve this problem by obtaining and installing the latest drivers for the system. Often people worry about data loss or file corruption because of this problem; however, this is almost never an issue once new driver is installed. If you need to install new drivers, you may need to boot into Safe Mode to remove the corrupt drivers. To access the Windows Safe Mode, press <F8> when you see “Starting Windows 95,” and then press <3> from the list of choices. You can click here for an alternate solution to the problem.

What is the DirectX release that Microsoft provides?

Microsoft releases DirectX updates that will install the latest DirectX drivers for the application you are trying to use. Several customers install this update, and they are very successful with the DirectX game that they are trying to play. Additionally, some software vendors provide this update with their latest applications. Visit Microsoft's DirectX Web site for the update!
