How do I add or remove a program that came with Windows® 95?

  1. Click on the Start button; choose Settings from the menu, and then choose Control Panel from the expanded menu.
  2. Double-click on the Add/Remove programs icon.
  3. Click on the Windows Setup tab.
  4. Click on the Details... button if you want to install specific aspects of a Windows component.
  5. Add or remove a check in the box next to the Windows component to install the feature.

How do I create a shortcut for an application in Windows® 95?

As you add applications to your computer, you may want to quickly access the application through the Windows® 95 Desktop. You can create shortcuts on your Windows® 95 Desktop that will allow you to double-click on the icon to begin application. To create an application shortcut, right-click on an empty area on the Windows 95 Desktop, and then choose New from the menu. Click on Shortcut from the expanded menu; you should see the Create Shortcut screen. Enter the command line for the application in the dialog box. If you do not know the name of the executable, you can click on the Browse... button to locate the appropriate file.

I expanded the Documents menu in the Start menu. Can I clear the documents from this expanded menu?

Within Windows® 95, you can access documents that have been recently accessed through the Documents menu. At a certain point, you may want to clear this menu. You can follow this procedure to clear this menu:

  1. Right-click on an empty area of the Windows® 95 Taskbar, and then choose Properties from the Menu.
  2. Click on the Start Menu Programs tab.
  3. Click on the Clear button in the Documents menu to empty the Documents menu.

How do I download and extract an executable (*.EXE) file in Windows® 95?

  1. Create a new folder.
  2. Download the file, and then move this file to the empty folder.
  3. Double-click on the executable file. This will expand the compressed files contained within you downloaded.
  4. View the README.TXT file for further information to create the disk for the file or driver.

My monitor is Energy Star compliant. How can I take advantage of these features in Windows® 95?

Depending on which AST system you own and the capabilities of your monitor, you can control energy-saver features of the monitor in Windows® 95. To adjust these features in Windows® 95, right-click on an empty area on the Windows® 95 Desktop, and then choose Properties from the menu. Now, click on the Screen Saver tab. You can enable and disable these features through this screen. Additionally, you can use the up and down arrows to set the amount of time to enable the energy saving features.

I upgraded to Windows® 95; why do I not hear my modem dial?

Often modem sounds will disappear after you upgrade to Windows® 95. To regain this feature, follow this procedure:

  1. Double-click on the speaker icon on the Windows® 95 Taskbar.
  2. Click on the Options menu, and then choose Properties...
  3. Make sure all options are checked under Show the Following Volume Controls, and then choose OK.
  4. Check the PC Speaker section of the mixer screen, and make sure you select the appropriate volume level.

 

If the speaker icon is not on the Taskbar, follow this procedure:

  1. Click on the Start button, and choose Settings.
  2. Choose the Control Panel from the expanded menu.
  3. Double-click on the Multimedia option.
  4. Choose the Show Volume Control on the Taskbar option.
  5. Click on OK to register the changes.

 

You can also use the following AT commands to adjust the modem's volume:

  • M0 - Speaker is always off
  • M1 - Speaker is on until carrier is detected (factory default)
  • M2 - Speaker is always on
  • M3 - Speaker is on after dialing until carrier is detected

Use the following commands in HyperTerminal to adjust the modem volume:

  • L0 - Volume lowest
  • L3 - Volume highest

You can also use 1 or 2 for intermediate ranges.

How do I copy a floppy disk in Windows® 95?

Rather than using the COPY command in a DOS session, access the My Computer icon. Click the right mouse button on the disk drive that contains the diskette you want to copy and then choose the Copy Disk... option from the menu that appears. Select the appropriate drives in the Copy From: and Copy To: boxes and press the Start button.

Note: You can also access the Copy Disk... option from the Explorer utility.
