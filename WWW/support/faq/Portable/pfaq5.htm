Why do I see double images on my AST PowerExec in DOS or Windows?

You may see double or mirrored images in DOS or Windows if your system is does not use the proper video drivers. You can use the PowerExec Video Drivers disk to correct this problem. If you do not have the PowerExec Video Drivers disk, you can create it by downloading the following file:

PEVID1.EXE PowerExec Video Drivers v1.01

Follow this procedure to change the video drivers:

  1. Exit all software, including Windows.
  2. Enter the following command to get into the WINDOWS directory:
    CD\WINDOWS
  3. Enter the following command:
    SETUP
  4. After SETUP determines your system configuration, use the arrow keys to select the Display Type field and press <Enter>.
  5. Use the arrow keys to scroll to the last selection, "Other video driver (Requires disk from OEM)." Insert your video driver disk and press <Enter>.
    NOTE: Make sure the System field has "MS-DOS' or "MS-DOS with APM." Sometimes Windows will select "AST Cupid 386 System." Your portable system is NOT a Cupid system, and this setting will cause problems.
  6. Choose one of the AST PowerExec video drivers from the list of drivers that appear on the screen. Your system will support 640x480x256 colors.
  7. Follow the on-screen directions to complete the video driver installation.

Running in 800x600 resolution, my mouse stays about 1 inch above the bottom of the screen on my Ascentia 950; what do I need to do?

Follow these steps to resolve this issue:

  1. Choose Display in Control Panel.
  2. Select the Settings tab.
  3. Select Change Monitor Type.
  4. Select “Standard Monitor Type Laptop Display Panel (800x600).”
  5. Choose OK, and then click Close.
  6. Choose 800x600 pixels on the Change Display Settings Screen,.
  7. Choose OK.

If the display doesn't change after you complete these steps, shut down the computer, reboot the computer, and then repeat steps 6 and 7.

These changes should resize the screen, and the mouse should function properly.

My Ascentia 950N does not have a full display in DOS mode; what should I do?

By default, the video display will have a reduced font in text mode that will produce a screen that uses the center portion of the display with a one inch margin on all sides. You can enable font expand in CMOS that will allow the computer to use a text screen that takes advantage of the full display area in DOS. Follow this procedure:

  1. Press <Fn+F2> from any screen.
  2. Tab to the Boot Option sub menu.
  3. Arrow to the Font Expand option.
  4. Use the left or right arrow keys to change the field to Enable.
  5. Press <Esc> to return to the Main Menu.
  6. Press <Enter> to save the changes and Exit.
  7. Close all open applications, and reboot the computer.

The change takes effect once the computer reboots.

Why does the clock not keep the correct time when I resume from suspend on my Ascentia J?

Enable Advanced Power Management in Windows 95. If you use Windows for Workgroups, then install the DOS Advanced Power Management driver.

My Ascentia P does not display a full screen using 640x480 resolution, why?

Update the Ascentia P BIOS to version 1.02. Listed below is the file you can download:

05584033.EXE Ascentia P System BIOS, Version 1.02

This BIOS incorporates the font expand feature. Press <Fn+F10>, from any window, to turn font expand on or off.
