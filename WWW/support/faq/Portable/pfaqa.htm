Dock ID

All Ascentia P Full docks come with an EEPROM with the same Electrical Identification programmed inside. If you use multiple docks with identical IDs on your notebook, then your notebook will use the same docked profile in Windows® 95, regardless of where you are docked. This is an advantage for using multiple docks that are configured the same. However, if you frequently dock the system to full docks that are configured differently and you want to maintain separate dock profiles, then you can program the Electrical ID of each dock to be different. This will result in Windows 95 creating separate dock profiles for each dock.

To change the electrical ID of your dock, download the following program:

FDAID.EXE

On the Ascentia 9xx, why does the floppy drive not work with NT 3.51?

To remedy the loss of floppy drive on the Ascentia 9XXN when running NT 3.51, boot the system to a DOS prompt. Enter the command SET4NT; this command will list the available parameters.

Execute the command again with the 1 switch:

SET4NT 1

Reboot the system and run NT. Windows NT will now recognize the A: drive when the system is running on battery power.

After the utility has been used to set the bit, the utility will not need to be used again unless the CMOS setup information is lost or corrupted.

How can I prevent the cursor from drifting on my Ascentia?

The trackstick controller has an embedded calibration routine to compensate for temperature variations within the system. Often the cursor will drift if you touch the trackstick in the calibration window. Since the calibration routine monitors the activity of the trackstick, you will disrupt the calibration settings. If there is no trackstick activity for eight seconds, the controller will initiate the calibration, which takes approximately one millisecond to complete. If you touch the trackstick during this one millisecond, the calibration routine will try to compensate for the voltage difference it has just detected, causing the cursor to drift.

To correct a drifting cursor, you can reset the device by invoking a SUSPEND <Fn+Esc> and RESUME <any key> operation. This change will reset the cursor without the user losing any information.

Additionally, if you use a cellular phone near an AST Ascentia 900N notebook that is running Windows 3.1, the cursor or mouse pointer may drift toward the edge of the screen. The issue is the result of the Ascentia 900N's lower board. This board may act as an antenna. As the system receives the cellular phone output signals it may feed the signals into the mouse control circuitry; as a result, the circuit may interpret that the track stick is being pushed, thus moving the mouse cursor.

The mouse cursor movement does not cause a problem in the Ascentia 900N system. The trackstick still operates normally, even when it is close to a source of interference, such as a cellular phone, cordless phone, or radio transceiver. Operate these devices at a distance of at least two feet away from the Ascentia 900N to prevent the trackstick from drifting.

How do I use "non-printer" hardware on my Ascentia 900N's parallel port?

When some third-party devices are connected to the Ascentia 900N parallel port, the software does not properly recognize the interrupt on LPT1. For example, the following error message may appear on the display:

IRQ 7 could not be found on LPT1

Older versions of software do not properly recognize the VLSI implementation of the super I/O chipset.

For other software conflicts, contact the third-party software manufacturer for the latest drivers that support the VLSI implementation.

When I boot the computer, I get a message "non system disk or disk error." What should I do?

Several factors can cause your system to display this message. First, check your floppy drive. If there is a disk in the drive, remove it and press any key.

If there is no disk in the drive or you continue to receive the message, you may have some damaged system files. Consider the following steps to restore the system files to your system's hard drive:

  1. Boot the system with Disk 1 of your DOS disks.
  2. Press <F3> to exit the DOS installation.
  3. Enter the following command at the A:\> prompt :
    SYS C:

This command should restore your system files, and allow your hard drive to boot.

You may see double or mirrored images in DOS or Windows if your system is does not use the proper video drivers. You can use the PowerExec Video Drivers disk to correct this problem. If you do not have the PowerExec Video Drivers disk, you can create it by downloading the following file:

PEVID1.EXE PowerExec Video Drivers v1.01

Follow this procedure to change the video drivers:

  1. Exit all software, including Windows.
  2. Enter the following command to get into the WINDOWS directory:
    CD\WINDOWS
  3. Enter the following command:
    SETUP
  4. After SETUP determines your system configuration, use the arrow keys to select the Display Type field and press <Enter>.
  5. Use the arrow keys to scroll to the last selection, "Other video driver (Requires disk from OEM)." Insert your video driver disk and press <Enter>.
    NOTE: Make sure the System field has "MS-DOS' or "MS-DOS with APM." Sometimes Windows will select "AST Cupid 386 System." Your portable system is NOT a Cupid system, and this setting will cause problems.
  6. Choose one of the AST PowerExec video drivers from the list of drivers that appear on the screen. Your system will support 640x480x256 colors.

Follow the on-screen directions to complete the video driver installation.

Why does my PC Card (PCMCIA) not work after I run MEMMAKER?

When you run MEMMAKER, it will try to configure your boot files for optimal use. However, it will often change the EMM386 line of your CONFIG.SYS and disable the memory range needed for PCMCIA use. AST systems require an exclusion of the entire D000 range. Consider the following syntax for excluding that range:

DEVICE=C:\DOS\EMM386.EXE NOEMS X=D000-DFFF

Some cards will not require you to exclude the entire range. Check with your PC card vendor if you need more memory.

Also, MEMMAKER often adds the HIGHSCAN switch to the EMM386 line. If you see this switch, remove it. Due to the added demands of power management in a portable system, AST recommends removing HIGHSCAN from the EMM386 line. In most cases, HIGHSCAN will try to force TSRs (terminate and stay resident programs) and other drivers into memory that are normally used by other notebook functions.

My Premium Exec / Advantage NB 386 notebook hangs after MS-DOS 6.2X has been installed. What should I do?

If the notebook hangs during the HIMEM.SYS extended memory text, follow this procedure:

  1. Turn on your system. As soon as you see the message, "Starting MS-DOS...," press <F5>.
    This should bypass your AUTOEXEC.BAT and CONFIG.SYS files.
  2. Enter the following command:
    PATH=C:\DOS
  3. To edit your CONFIG.SYS file, enter the following command:
    EDIT CONFIG.SYS
  4. Search the CONFIG.SYS file for the line that contains HIMEM.SYS. When you find the line, you will need to add the "/TESTMEM:OFF" switch.
    Note: The HIMEM.SYS line should be similar to the following example:
    DEVICE=C:\DOS\HIMEM.SYS /TESTMEM:OFF
    This switch disables the memory test, which allows your system to boot without hanging.
  5. Save the file, exit the MS-DOS Editor, and reboot your system.

How do I get a 3Com 3C509B Network Card to work in the PowerStation III or PowerStation Pro?

The configuration program for the adapter says that "ALL THE PORTS ARE BEING USED" or "THIS INTERRUPT IS IN USE, WOULD YOU STILL LIKE TO USE IT?" If you answer Yes to either of these errors and then run diagnostics on the card, the interrupt will fail.

Consider the following procedure to resolve the problem:

  1. Choose the port addressing and interrupt (IRQ) you would like to use. In general, port address 300 and IRQ 9 work best.
  2. Save the setup and DO NOT run diagnostics on the card!
  3. Exit the program and reboot the system.
  4. Reenter the configuration program, and run the diagnostics on the card. It should now pass all the tests.

My notebook will not power up when docked in the PowerStation Pro. What should I do?

The PowerStation Pro has a plunger switch to establish a complete connection between the notebook and the PowerStation Pro. If the back port-access door has been removed, the plunger will not be depressed. As a result, the PowerStation Pro will not power up. The back port access door must be in place and in the closed position.

How do I maintain optimal battery performance?

There are several factors that can control that can determine the performance and life of your battery.

If you run the system on AC power while the battery is still installed, you may shorten the life of the battery. The battery will continue to charge if you leave the battery inside the system and run the computer on AC power. If you do not need to use the mobile features of the system, then remove the battery from the computer and run the system on AC power.

If your portable system has a Nickel Cadmium (NiCad) or Nickel Metal Hydride (NiMH) battery, you should perform a full battery discharge after every one to two months. NiCad and NiMH batteries will develop a "memory" if you do not regularly perform this procedure. Although the "memory" effect are not as dramatic on NiMH batteries as the effects on a NiCad batteries, the "memory" effects occur in both battery types.

Follow this procedure to perform a deep battery discharge:

  1. Turn the system off.
  2. Charge the system until your battery light disappears or indicates a trickle charge.
  3. Disconnect the AC adapter from the system.
  4. Turn on the system and use the it until the system shuts down. To expedite the discharge, you may want to enter System Setup and choose Max Performance in the Power Management settings.
  5. Repeat step four (4) until the system will not power up on battery power.
  6. With the system off, connect the AC adapter to the system and charge the battery for approximately one to two hours.

Your battery should be refreshed and fully charged.

If your needs require that you use the battery extensively, you may want to consider purchasing additional batteries as an insurance measure. Many applications are disk-intensive, and can quickly drain a battery. Some batteries included with notebook systems can use an external battery charger; this is another wise investment if you plan to run the system on battery power for extended periods.

Can I upgrade the internal modem for the Advantage! NB 486SX/33 to a 14.4Kbps or higher?

This notebook uses an 8250 UART chip. The 8250 UART limits the speed of the serial port to a maximum baud rate of 9600bps in Windows, and will not allow a 14.4Kbps or faster external modem to operate reliably at a speed faster than 9600bps. The fastest modem AST recommends is a 9600bps modem.

Is the processor upgradable on any of the AST notebooks?

The processors on all AST notebooks are soldered onto the system board of the computer. The processor cannot be upgraded or replaced.

Do I need to make any changes to my Ascentia 910N if I install Windows 95?

The BIOS for the Ascentia 910N does not support the Windows 95 Plug-and-Play and Advanced Power Management features. You need to disable the Advanced Power Management option in Windows 95.

The steps to disable Advanced Power Management in Windows 95 are listed below:

  1. Click Start, Settings, and Control Panel.
  2. Double-click the System icon.
  3. Click the Device Manager tab.
  4. Double-click the plus sign to expand the System Devices branch.
  5. Double-click the Advanced Power Management option.
  6. Remove the check next to the box that labeled, “Enable Advanced Power Management support.”
  7. Click OK.

I have Windows 95 on an Ascentia 910N and my PCMCIA Ethernet card is hanging; what should I do?

Random data may be received when Windows 95 polls the VLSI controller for power management status. This random data may lock up the system. To resolve this issue, disable power status polling. Follow the steps below to disable power status polling:

  1. Click the Start button, point to Settings, and then click Control Panel.
  2. Double-click the System icon.
  3. Click the Device Manager tab.
  4. Double-click the plus sign to expand the System Devices branch.
  5. Click Advanced Power Management, Properties, and Settings.
  6. Place a check in the box labeled “Disable Power Status Polling.”
  7. Click OK until you return to the Control Panel.
  8. Click Yes if prompted to restart your computer.

Why doesn't my keyboard or mouse work in Windows for Workgroups?

The Ascentia 900N system keyboard may hang or fail to operate properly under Windows for Workgroups, version 3.11. This problem is caused by the way the keyboard controller circuit is implemented on some systems. Install a separate keyboard driver which ignores the extra interrupt signals generated by these keyboard controllers. Windows normally uses an internal "Virtual Keyboard Driver" or VKD.

To install the new VKD.386 driver:

  1. Download VKD386.EXE. This self-extracting file will create the driver.
  2. Exit Windows.
  3. Copy the file VKD.386 to your WINDOWS\SYSTEM directory.
    Consider this example:
    COPY A:\VKD.386 C:\WINDOWS\SYSTEM
  4. Change to the Windows directory.
  5. Using the MS-DOS EDIT command to edit your SYSTEM.INI file:
    EDIT C:\WINDOWS\SYSTEM.INI
  6. Search for the following line: KEYBOARD=*VKD. While in MS-DOS EDIT, press <F3>, and enter "*VKD" to find it.
  7. Change this line to read like the following example:
    KEYBOARD=VKD.386
  8. Save the SYSTEM.INI file. In MS-DOS EDIT, press <ALT>,<F>,<S>,<Alt>,<F>,<X>.
  9. Reboot your system and run Windows for Workgroups 3.11. The keyboard and mouse will now operate properly.

I get a “Bat1 Gauge Unsure!" message on my Ascentia P computer, how do I fix this problem?

This warning appears whenever the battery gauge cannot accurately measure the battery charge because of too many partial discharges of the battery. A partial discharge occurs when the AC adapter is connected before the battery charge is fully depleted. If this occurs too many times, the battery gauge cannot fully calibrate itself. Follow the steps below to resolve this issue:

  1. Turn off power management by pressing <Fn+F5>.
  2. Disconnect the AC adapter.
  3. Keep the unit's power on until the battery is completely discharged. If you continue to work, make sure you save your work once you received the battery-low warning. The battery is fully discharged when the computer beeps three times and the battery indicator light flashes twice per second. Your computer automatically goes into suspend mode.
  4. Fully charge the battery once your computer's power automatically turns off.

My Ascentia P reports the floppy drive as a 5.25 inch drive, rather than a 3.5 inch drive; what do I need to do?

To resolve this issue on the Ascentia P, perform the following steps:

  2. Access the security menu, and check “Diskette access.” If it is set to Administrator change it to User.
  3. Select Not Installed, at the “Disk A:” option.
  4. Save the changes and exit the CMOS Setup.
  5. Reboot the computer into the Safe Mode by pressing <F8>. When you see the message "Starting Windows 95.”
  6. Select option 6, “Safe mode command prompt only.”
  7. Enter the following command to access the A: drive:
    DIR A:
  8. Choose abort.
  9. Reboot the computer, and access the CMOS System Setup by pressing <F2>.
  10. Select 1.44 or 3.5 inch drive, at the Disk A: option.
  11. Save the changes and Exit.
  12. Reboot the computer, and the floppy drive should work properly.

This BIOS incorporates the font expand feature. Press <Fn+F10>, from any window, to turn font expand on or off.

I get a DSOUND.DLL error message whenever I start up a game on my Ascentia A system; what should I do?

Download and install driver version 1.80 or later for the Crystal CS4232/6 audio chipset. The file name is W95US201.ZIP, and it is available at the following URL:

http://www.crystal.com/drivers/crysoft.htm

The installation instructions are contained in the README.TXT file included with this driver.

The mouse drifts on my Ascentia A; what do I need to do?

The following steps will resolve this issue:

  1. Right-click on the Touchpad icon on the Start Tray (taskbar).
  2. Click on "Settings."
  3. Remove the check from "Turn off edge motion" in the Touchpad Enhancements window.
