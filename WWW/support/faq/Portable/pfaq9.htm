I have a ProStation Multimedia docking station, and an Ascentia 900N; why will the Toshiba CD-ROM not work in the ProStation?

Obtain the latest Toshiba CD-ROM drivers for the ProStation Multimedia Docking Station. Listed below, is the file you can download:

807032.EXE Toshiba 4x CD-ROM driver(s) v.2.00

I can not hear sound when I am playing audio CDs on my Ascentia P computer; what changes do I need to make?

Follow these steps to reset the volume control for the CD-ROM drive:

  1. Double-click on the volume icon located on the right corner of the task bar.
  2. Select the CD Balance option.
  3. Set the volume to high.
  4. Place a check mark in the box next to the Mute option.
  5. Remove the check from the Mute Box.
  6. Shut down and restart Windows 95.

I get a DSOUND.DLL error message whenever I start up a game on my Ascentia A system; what should I do?

Download and install driver version 1.80 or later for the Crystal CS4232/6 audio chipset. The file name is W95US201.ZIP, and it is available at the following URL:

http://www.crystal.com/drivers/crysoft.htm

The installation instructions are contained in the README.TXT file included with this driver.

The mouse drifts on my Ascentia A; what do I need to do?

The following steps will resolve this issue:

  1. Right-click on the Touchpad icon on the Start Tray (taskbar).
  2. Click on "Settings."
  3. Remove the check from "Turn off edge motion" in the Touchpad Enhancements window.
