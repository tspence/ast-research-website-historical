I'm trying to reinstall Windows on my Advantage! Explorer or Bravo NB and it hangs on the second or third diskette. What do I do?

When you run Windows SETUP, select Custom Setup. When SETUP displays your system's information, make sure VGA is in the Display Type field. The Windows installation should now complete successfully.

After you complete the Windows installation, you need to install the proper drivers, which you can find on your Notebook Utilities Disk. This disk is one of the disks you made when you received the "MAKE YOUR DISKS NOW!" message. If you do not have this disk, you can create it by downloading and running the following file:

BNB_VID.EXE Bravo NB/Adv! Explorer video drivers 2.6

Once you create the disk, you can follow this procedure to change your system's video drivers:

  1. Double-click on the Windows Setup icon under the Main Program Group.
  2. Click once on the Options selection on the menu bar.
  3. Hold down the mouse button and scroll down to the Change System Settings... selection.
  4. Release the mouse button to view the selection screen.
  5. Click on the down arrow to the right of the Display line.
  6. Release the mouse button tobring up a listing of video drivers installed on your system.
  7. Scroll to the bottom of the list and select "Other Display (Requires Disk from OEM)."
  8. Insert the drivers diskette and change the path to the specified drive.
  9. Choose one of the drivers that includes "*Chips 655XX." For example, "*Chips 655XX Linear."

After you load the drivers, restart Windows.

Do I need to make any changes to my Ascentia 910N if I install Windows 95?

The BIOS for the Ascentia 910N does not support the Windows 95 Plug-and-Play and Advanced Power Management features. You need to disable the Advanced Power Management option in Windows 95.

The steps to disable Advanced Power Management in Windows 95 are listed below:

  1. Click Start, Settings, and Control Panel.
  2. Double-click the System icon.
  3. Click the Device Manager tab.
  4. Double-click the plus sign to expand the System Devices branch.
  5. Double-click the Advanced Power Management option.
  6. Remove the check next to the box that labeled, “Enable Advanced Power Management support.”
  7. Click OK.

I am running Windows NT 3.51 on my Ascentia 950N and I receive kernal trap errors; what should I do?

To eliminate trap errors created while on battery power, run Windows NT in turbo mode by pressing <Fn+F4>. Your other option is to use the system with AC power.

I am running Windows 95 on my Ascentia 950N, and my floppy drive is running in MS-DOS Compatibility Mode; what should I do?

Consider the following procedure to eliminate the MS-DOS Compatibility mode on the floppy drive:

  1. Double-click the Add New Hardware icon from Control Panel, and then click on Next.
  2. Select No to the question, "Do you want Windows to search for your new hardware?"
  3. Click the Next button, and then double-click the Floppy disk controller option.
  4. Double-click “Standard floppy disk controller”; then click the Next button.
    NOTE: Ignore the message about a device conflict.
  5. Click on Finish.
  6. Double-click the System icon From Control Panel, and click the Device Manager tab.
    NOTE: The Standard floppy disk controller is displayed with a red X.
  7. Double-click on “Standard floppy disk controller.”
  8. Check the hardware profile options under the Device Usage area. Depending on your configuration, these options are:
    Docked
    Undocked
  9. Click on the Resources tab, and click on the Set Configuration Manually button.
  10. Select Basic Configuration 1, and click OK at bottom of the screen.
  11. Return to the Device Manager tab, and click the plus sign to expand the System Devices option.
  12. Follow the Windows Restart procedure as directed by Windows® 95.

Once the system restarts, the floppy drive should be in 32-bit mode.

My Ascentia 950N hangs when Windows 95 is shutting down; what should I do?

Enable Advanced Power Management in CMOS setup and in Windows 95.

Follow these steps to enable Advanced Power Management in the BIOS:

  1. Press <Fn+F2> from any screen.
  2. Arrow down to the Advanced Power Management option.
  3. Use the left or right arrow to change the field to Enable.
  4. Press <Esc> to return to the Main Menu.
  5. Press <Enter> to save the changes and Exit.

Follow the steps below to enable Advanced Power Management in Windows 95:

  1. Click Start, Settings, and Control Panel.
  2. Double-click the System icon.
  3. Click the Device Manager tab.
  4. Double-click the plus sign to expand the System Devices branch.
  5. Click Advanced Power Management, Properties, and Settings.
  6. Place a check in the box labeled “Enable Advanced Power Management Support.”
  7. Click OK.

I can’t get Microsoft Office to install under Windows for Workgroups on my Ascentia J; what should I do?

Edit the CONFIG.SYS file and REM out the CardWiz lines. Reboot the system, and Microsoft Office should perform a complete installation. After Microsoft Office is installed, edit the CONFIG.SYS, remove the REM statements, and reboot the system.

I get a DSOUND.DLL error message whenever I start up a game on my Ascentia A system; what should I do?

Download and install driver version 1.80 or later for the Crystal CS4232/6 audio chipset. The file name is W95US201.ZIP, and it is available at the following URL:

http://www.crystal.com/drivers/crysoft.htm

The installation instructions are contained in the README.TXT file included with this driver.

The mouse drifts on my Ascentia A; what do I need to do?

The following steps will resolve this issue:

  1. Right-click on the Touchpad icon on the Start Tray (taskbar).
  2. Click on "Settings."
  3. Remove the check from "Turn off edge motion" in the Touchpad Enhancements window.
