AT Commands

This modem was designed to respond to commands based on Rockwell International's AT command set. You can obtain copies of the AT command sets from Rockwell International at the following site:

http://www.nb.rockwell.com/ref/at_commands.html

Listed below is the files that you can download from the site:

1048rev4 Acrobat PDF file

You will need Adobe Acrobat to read this file.

You can view the HTML version of the manual at the following Web site:

http://www.nb.rockwell.com/ref/1048PR3a.html

Section 3 of this manual describes the AT commands for the modem, and Section 4 describes the S-Registers.
