Initialization Strings

Modems generally use a series of AT commands, called initialization strings, to provide a good connection to another modem. Initialization strings are personal, and your best bet will be to contact your Internet service provider (ISP) to help you develop the best string that meets your needs.

Listed below is a common initialization string for this modem:

AT&F&C1&D2&K3W1

The following string will disable error correction and data compression:

AT&F&C1&D2/Bn%C0

Note that /Bn disables error correction, while %CO disables data compression.

Often ISPs will have specific settings for you to select within a communication applet. The selection you make within these applets automatically configures the appropriate initialization string for you to use. For most communication applets you can select a Silicom or Generic Hayes Compatible modem. For America Online you can select a Hayes Compatible V2.5 (default); Prodigy users can select a Hayes Compatible 288, and CompuServe users can select BOCA MV28KINT/EXT.
