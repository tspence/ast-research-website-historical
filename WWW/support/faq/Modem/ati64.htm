Windows 95® Drivers

This modem shipped with systems that were designed to function in a Windows® 95 operating system. AST also installed backup copies of the ATI modem drivers in the Create System Disk Utility/AST Disk Utility and in the CAB files (C:/WINDOWS/OPTIONS/CABS) on the hard drive. You can also download drivers from our Web site. Listed below is the file that you can download:

ATIATTDR.EXE ATI/AT&T Modem Drivers Win95, v1.0

Follow this procedure to create the driver diskette:

1. Download ATIATTDR.EXE.

2. Save the file to an empty (e.g., TEMP) directory.

3. Double-click on the ATIATTDR icon.

4. Double-click on the INSTALL icon.

5. Insert a blank floppy disk in the floppy drive.

6. Follow the on-screen procedures to create the diskette.

Other Operating Systems

AST intended, designed, and manufactured the systems that use this modem to function with a Windows 95 operating system. As a result we do not have other drivers for this modem. However, that does not mean the drivers are not available.

This ATI modem uses the same chipset as a Boca (AT&T chipset) modem. As a result, you can use the DOS/Windows 3.x driver as the Boca (AT&T chipset). The DOS driver is contained within BATTDRV.EXE. You can run MDMSETUP.EXE on the driver disk to install the DOS/Windows 3.x drivers. Additionally, you may need to access System Setup and disable the Plug-and-Play operating system (PnP OS) within the Advanced Menu.

AST does not have Windows NT™, OS/2 Warp, or Linux drivers for this modem.
