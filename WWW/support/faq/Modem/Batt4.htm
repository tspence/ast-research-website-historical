Drivers

This modem shipped with systems that were designed to function in a Windows® 95 operating system. AST also installed backup copies of the Boca modem drivers in the Create System Disk Utility/AST Disk Utility and in the CAB files (C:/WINDOWS/OPTIONS/CABS) on the hard drive. You can also download drivers from our Web site. Listed below is the file that you can download:

BATTDRV.EXE Win95 v4.27 Boca AT&T Modem

Follow this procedure to create the driver diskette:

1. Download BATTDRV.EXE.

2. Save the file to an empty directory (e.g., TEMP).

3. Double-click on the BATTDRV icon in the directory where you saved the file; this will expand the compressed files.

4. Double-click on the EXTRACT icon.

5. Follow the on-screen procedures in the DOS box to create the driver disk.

Other Operating Systems

AST intended, designed, and manufactured the systems that use this modem to function with a Windows 95 operating system. As a result we do not have other drivers for this modem. However, that does not mean the drivers are not available.

AST did include Windows® 3.1/DOS drivers on the driver disk you can make for the system, as well as within BATTDRV.EXE. You can run MDMSETUP.EXE on the driver disk to install the DOS/Windows 3.x drivers. Additionally, you may need to access System Setup and disable the Plug-and-Play operating system (PnP OS) within the Advanced Menu.

AST does not have Windows NT™ drivers for this particular modem; however, you may want to try the Boca Research FTP site.
