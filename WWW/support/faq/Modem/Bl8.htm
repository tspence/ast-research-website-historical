Reinstallation Procedures

Use the following procedures to install and configure the Boca Lucent MV.34AI FDSP34AI modem:

Configuring the COM Ports:

1. Enter the Ports section of System Setup. To enter System Setup, press <Ctrl+Alt+Esc> during a cold boot.

2. Ensure that the serial ports are arranged as follows:

Serial port 1: 3F8
Serial port 2: 3E8

3. Exit Setup and save changes.

Installing the COM Port drivers:

1. Access the Add New Hardware icon when you re-enter Windows.

2. Do not let Windows® 95 automatically detect the hardware.

3. Choose Ports (COM & LPT) and then Standard.

4. Select MV.34AI FDSP34AI and click Next.

NOTE: If the port is not listed, make the driver diskette from the Create System Disks program. After you create the driver disk, restart the procedure, but choose Have Disk instead of Standard COM port.

5. Follow the on-screen instructions to complete the installation procedure, and restart the computer when prompted.

6. Access the Device Manager from the System icon in Control Panel, and select the Ports branch.

7. Double-click on MV.34AI FDSP34AI.

8. Ensure that the resources are configured as follows:

Input/Output Range: 02F8-02FF
Interrupt Request: 3

9. Click OK to close Device Manager, and restart the computer if prompted.

Installing the Modem Drivers:

1. Double-click on Modems in Control Panel.

2. Do not let Windows automatically detect your modem.

3. Select 28.8Kbps V.34 MV.34A under Standard Modem Types.

NOTE: If this option is not available, choose the Have Disk option and insert the modem driver disk. Select the 28.8Kbps V.34 MV.34A modem.

4. Install the modem on COM2, and follow the on-screen instructions to finish the installation.

5. Choose C:/WINDOWS/OPTIONS/CABS when prompted for the location of the Windows 95 files.
