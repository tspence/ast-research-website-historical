Reinstallation Procedures

You can use the following procedures to reset the ATI 300 Series modem:

Removing the existing drivers:

1. Double-click on the System icon in the Windows Control Panel.

2. Click on the Device Manager tab.

3. Remove all entries from the Ports (COM & LPT) and Modems sections.

4. Restart the computer and enter System Setup. In the Advanced section, ensure that Configuration Mode is Manual and that the Serial Port 2 address is COM3.

5. Exit System Setup and Save Changes.

Reinstalling the COM Ports:

1. Choose Add New Hardware from the Windows® 95 Control Panel.

2. Do not let Windows automatically detect the hardware.

3. Insert the driver diskette and click Have Disk.

4. Specify C:/WINDOWS/OPTIONS/CABS and select the 300 Series Fax-Modem.

5. Restart the computer when prompted.

6. Access the Ports (COM & LPT) branch of Device Manager and access the properties of the 300 Series Fax-Modem. Ensure that the settings are as follows:

Input/Output Range: 02F8-02FF
Interrupt Request: 3

7. Click OK or Close and restart the computer.

Reinstalling the modem drivers:

1. Access the Modems section of Control Panel. When the Add New Modem Wizard begins, do not let Windows automatically detect the modem.

2. Click on Have Disk and specify C:/WINDOWS/OPTIONS/CABS. Choose 300 Series Fax-Modem.

3. Select 300 Series Fax-Modem (COM2) when you specify the port address.

4. Restart the computer.
