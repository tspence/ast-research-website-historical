AT Command Reference Table

Dial Commands Ampersand Commands S-Registers

Dial Commands

0-9 Dialable digits
* Auxillary tone-dial digit
# Auxillary tone-dial digit
T Tone dialing
P Pulse dialing
R Call an originate-only modem
, Pause
; Remain in command mode after dialing
" Dial Alpha Phone #s
W Wait for second dial tone
@ Wait for answer
! Flash switch hook

 

Ampersand Commands

&An n=0 Disable ARQ Result Codes
  n=1 Enable ARQ Result Codes
  n=2 Modulation Codes
  n=3 Extra Result Codes
&Bn n=0 Floating DTE Speed
  n=1 DTE Speed fixed when ARQ
  n=2 CD Always On
&Cn n=0 Modem Controls CD
&Dn n=0 Ignore DTR
  n=1 Reserved
  n=2 DTE Controls DTR
&F   Load Factory Configuration
&Gn n=0 No Guard Tone
  n=1 550Hz Guard Tone
  n=2 1800Hz Guard Tone
&Hn n=0 Disable TX Flow Controler
  n=1 CTS
  n=2 Xon/Xoff
&Pn n=0 N. American Pulse Dial
  n=1 U.K. Pulse Dial
&Rn n=0 CTS follows RTS
  n=1 Ignore RTS
  n=2 RX to DTE/RTS high
&Sn n=0 DSR Always on
  n=1 Modem Controls DSR
  n=2 Pulse DSR CTS=CD
  n=3 Pulse DSR
&Tn n=0 End Test
  n=1 Analog Loopback
  n=3 Digital Loopback
  n=4 Grant Remote DLB
  n=5 Deny Remote DLB
  n=6 Remote Digital Loopback
  n=7 Remote DLB with Self Test
  n=8 ALB with Self Test
&W   Store Configuration
&Xn n=0 DCE Synchronous Clock

 

S-Registers

S0 Ring to answer on
S1 Counts # of rings
S2 Escape character
S3 Carriage Return
S4 Line Feed
S5 Backspace
S6 Wait Time/Dial Tone (seconds)
S7 Wait Time/Carrier (seconds)
S8 Comma time
S9 Carrier Detect Time
S10 Carrier Loss Time
S11 Dial Tone Spacing
S12 Escape Code Time
S13 Bit Mapped
  1 = Reset on DTR Loss
  2 = Do Originate in Auto Answer
19 Inactivity Timeout
20 Reserved
21 Break Length
22 Xon Character
23 Xoff Character
24 DSR Pulse Time
25 Reserved
26 RTS/CTS Delay Time
27 Bit Mapped
  1 = V21 Mode
  2 = Disable TCM
  4 = Diable V32
  8 = Disable 2100Hz
  16 = Disable MNP Handshake
  32 = Disable V.42 Detect Phase
  64 = Reserved
