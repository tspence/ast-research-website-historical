Reinstallation Procedures

You can use the following procedure to reset the modem in your system:

1. Double-click on the System icon in the Windows Control Panel.

2. Click on the Device Manager tab.

3. Remove all entries from the Ports (COM & LPT) and Modems sections.

4. Restart the computer and enter System Setup. In the Advanced Menu, ensure that Configuration Mode is set to Manual and that the Serial Port 2 Address is COM3. Exit and save changes.

5. Allow the system to boot into Windows® 95. Windows 95 will automatically detect the ports and the modem. The system is configured properly if you find the following settings listed in Device Manager:

Modems:
Silicom Modem Master 6000 PnP

Ports (COM & LPT):
Communications Port (COM1)
Communications Port (COM3)
Printer port (LPT1)

Note that COM2 is not listed. The Plug-and-Play modem configures its own virtual COM2 port.
