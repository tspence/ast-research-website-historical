Reinstallation Procedures

1. Double-click on the System icon in the Windows Control Panel.

2. Click on the Device Manager tab.

3. Remove all entries from the Ports (COM & LPT) and Modems sections.

4. Restart the computer and enter System Setup. In the Advanced Menu, ensure that Configuration Mode is set to Manual and that the Serial Port 2 Address is COM3. Exit and save changes.

Windows® 95 will automatically detect the ports and the modem. The system will be configured properly if the following is listed:

Modems:
FDVSP Modem

Ports (COM & LPT):
Communications Port (COM1)
Communications Port (COM3)
Printer port (LPT1)

Note that COM2 is not listed. The Plug-and-Play modem configures its own virtual COM2 port.

If the modem still fails to work, you may need to manually install the modem drivers:

1. Make the driver diskette from one of the following resources:

AST Disk Utility

Create System Disks

BROCKDRV.EXE

Notes: If your system came bundled with the Boca (Rockwell chipset) modem, then AST included either the AST Disk Utility or the Create System Disk Utility. If you use the AST Disk Utility then you should select FDVSP34IG from the list to make the driver disk.

2. Check in Device Manager. Is there a section titled Other Devices? If so, highlight RC144AxF, choose Properties, and click on the Driver tab.

If RC144AxF is not listed in the Other Devices section, the modem is set up properly. You can skip the rest of the installation procedure.

3. Click the Change Driver button. If you are prompted to select the hardware type, choose Modem.

4. Click the Have Disk button and insert the Boca Rockwell driver disk in the floppy drive. Several files will be copied.

5. When prompted for CAB19, enter in the following directory:

C:/WINDOWS/OPTIONS/CABS
