Drivers

This modem shipped with systems that were designed to function in a Windows® 95 operating system. AST also installed backup copies of the Silicom modem drivers in the Create System Disk Utility and in the CAB files (C:/WINDOWS/OPTIONS/CABS) on the hard drive.

You can also obtain drivers from the following Web site:

http://www.silicom-multimedia.com/mm6000.htm#Latest Drivers

Other Operating Systems

AST intended, designed, and manufactured the systems that use this modem to function with a Windows 95 operating system. As a result we do not have other drivers for this modem. However, that does not mean the drivers are not available.

You can find Windows 3.x drivers on the following Web site:

http://www.silicom-multimedia.com/mm6000.htm#Latest Drivers
