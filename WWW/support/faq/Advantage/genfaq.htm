  2. VEBEDSK2.EXE -- Print Advantage! Manuals W/95 v1.0 (2 of 4)

These files contain a README document that will explain how to create the disk set. After you create the four installation disks, install them like any other application..

Why do I receive the message "Not an AST Computer" when I use the Hard Disk Restore floppy on my Advantage!® 4075p?

On the original release of the Advantage!® 4075p, there was an option in the AUTOEXEC.BAT file of the Hard Disk Restore that verified the system was an AST unit. This security feature was not added to the BIOS; consequently, the system returns with the message "Not an AST Computer." You can correct this problem by editing the AUTOEXEC.BAT file on the Hard Disk Restore floppy and removing the CHECKAST line.

How do I restore my Advantage! system to its default configuration?

From time to time, you may find it necessary to restore your computer to its factory configuration. AST has simplified this process through the "AST-SAVE" feature. Before you restore the default configuration, backup any files that you have saved on the system. Follow this procedure to restore the factory configurations:

  1. Exit Windows.
  2. Enter CD\AST-SAVE at the DOS prompt.
  3. Make sure that you have the "AST-SAVE" directory on your hard drive (C:\) if you receive an "Invalid Directory" error. You can also perform the restore from the "System Configuration Files" diskette that you created with the AST Disk Utility.
  4. Enter XCOPY *.INI C:\WINDOWS from the \AST-SAVE directory. Press <Y> to confirm you want to overwrite the old files.
  5. Enter XCOPY *.GRP C:\WINDOWS. Press <Y> to confirm you want to overwrite the old files.
  6. Restore the system configurations by entering COPY *.SYS C:\. Press <Y> to confirm you want to overwrite the old files.
  7. Enter XCOPY *.BAT C:\. Press <Y> to confirm you want to overwrite the old files.
  8. Reboot your computer for the changes to take effect.

For more information on this topic, search the ASK AST on-line help utility (Restoring original settings).

How do I create my master disks with the AST Disk Utility (ASTDU) or the Create System Disks utility?

AST includes disk images of the software that shipped with your computer. For the majority of AST systems, you can use the AST Disk Utility (ASTDU) to create your master set of disks. Follow this procedure to create disks with the ASTDU:

  1. Choose System Tools from the AST Works menu.
  2. Click on the ASTDU icon.

    3. Note: On systems that ship with a CD-ROM, you can also run the application from the \ORIGINAL\ASTDU directory on your AST Works CD.

  3. Select the application you want to create the master disks for. Click on the Click Here to Make Diskettes button. You will see a list of software that was pre-installed on your machine.
  4. Follow the procedures in the ASTDU to make the disk.
  5. Choose Yes if you want to remove the backup information from the ASTDU.

The ASTDU will automatically default to the next application when you finish making a set of disk. Press Skip to go to the next selection, or press Cancel if you want to quit the AST Disk Utility.ou can learn more about the AST Disk Utility by clicking on the Help button in ASTDU's opening screen.

On new Advantage! systems the AST Disk Utility has changed to the Create System Disks utility. To access this utility, make the following selections in Windows® 95:

  • Start
  • Programs
  • Accessories
  • System Tools

With the Create System Disks utility, you can make a variety of driver disks and copies of the original software that shipped with your AST computer. You can view the disks that have already been made for the system in a dialog box at the bottom of the Create System Disks utility.

How do I install a Windows 3.1 printer driver on my AST computer?

Before you can print from Windows applications, connect a printer to your computer's printer port, and then install a printer driver. The printer driver provides Windows with important information about your printer's configuration. There are three options to consider when you install a printer. You need to choose one of the following options:

  • Use a driver provided by the printer manufacturer.
  • Choose one of the printer drivers included with Windows 3.1.
  • Select a driver that your printer can emulate.

Follow this procedure to install a printer driver from the printer manufacturer's disk:

  1. Click on the System Tools button.
  2. Click on the Control Panel icon.
  3. Double-click on the Printers icon in the Windows Control Panel.
  4. Click on the Add>> button on the Printers screen.
  5. Insert the printer driver diskette in the floppy drive.
  6. Click on the Install... button
  7. Enter "A:\" in the "Insert unlisted, updated, or vendor-provided printer driver in:" field.
  8. Follow any step-by-step procedures included with printer driver.

Be sure to follow any documentation that you received with your printer before you follow this procedure. For more information on the options and capabilities of your printer, please consult your printer's reference manual.

Follow this procedure to install a printer driver that is included with Windows 3.1:

  1. Click on the System Tools button.
  2. Click on the Control Panel icon.
  3. Double-click on the Printers icon in the Windows Control Panel.
  4. Click on the Add>> button on the Printers screen.
  5. Click to highlight the printer you want to install in the "List of Printers:" dialog box.
  6. Click on the Install... button.

If your printer did not include a printer driver diskette and it is not listed in the "List of Printers:" dialog box, then your printer may emulate another common printer. Consult the documentation that shipped with your system for more information on which printer driver emulation you should select.

If you are connecting a PostScript printer, choose the name of the printer, not PostScript Printer from the list. If you are installing a laser printer that emulates PostScript language or uses PostScript cartridges, please consult your printer's documentation and your Microsoft Windows 3.1 User's Guide.

I just received a DOS 261 error message. What does this message mean?

The DOS 261 error is related to the AST GURU application. AST designed the GURU program to protect users from accidentally damaging important system files. This program saves all changes that you may make to the .INI files, icons, file paths, etc. The program is designed as a backup utility to protect against potential catastrophe.

The 261 error occurs when the system attempts to make a copy of GURU.INI while GURU.INI is already being accessed. Normally, this error will also arise when the disk that you use to make the GURU backup disk is pre-formatted. You need to format the disk on the system, and use this disk in the future for further updates. This error is not a serious problem that will harm your system. You can ignore the error and continue; it will not affect the backup.

You can also disable the GURU utility using REM statements in your AUTOEXEC.BAT. Once you REM the GURU line in the AUTOEXEC.BAT, you will not encounter the GURU utility during boot anymore; however, the program will still be active. Should you ever need to access GURU, you can enter "GURU.EXE" at the Run... line on the File menu.

Why am I unable to use 32-bit disk access on my Advantage! computer?

On some Advantage! systems, the IDE controller supports the primary hard drive and the CD-ROM drive. Systems that have an IDE CD-ROM drive attached to the same controller as the hard drive cannot properly utilize 32-bit access. Windows cannot discern between devices connected to one controller; there is no way to split up 32-bit capability between two dissimilar devices (e.g. a hard disk and a CD-ROM drive).

On some of the older Advantage! systems, the CD-ROM drive functioned from a controller on the sound card; they were not attached to the hard drive controller. This configuration allows you to enable 32-bit access for the hard drive without affecting the CD-ROM drive.

Although you cannot enable 32-bit disk access features in Windows 3.1, you can take advantage of the 32-bit operating system features in Windows® 95.

Another option is to install a third-party EIDE controller card. Once you install the controller card, you can place the CD-ROM drive on a controller that is independent of the hard drive(s).

After I play some of my DOS-based games, my system loses some of the system settings; what happened?

There are several games on the market that will reset the CMOS. A solution to this problem is to exclude the C000 to CFFF range from the EMM386 device line in the CONFIG.SYS file. You can consider this example:

    DEVICE=C:\DOS\EMM386.EXE X=C000-CFFF

This switch prevents programs from accessing the section of memory that the BIOS uses for shadowing.

How do I add a second hard drive to my Advantage! computer?

On several Advantage! systems, the single IDE controller on the system board already supports a maximum of two IDE devices, the primary hard drive and the CD-ROM drive. If your system has one controller, you need to purchase an Enhanced IDE (EIDE) controller to add a second hard drive. Depending on the architecture of the system, you may be able to use an ISA or PCI-based controller card. You can find most brands of EIDE controller cards at your local computer store.

Another factor to consider when you want to install another hard drive is the BIOS. On some older Advantage! systems, the BIOS does not support Logical Block Addressing (LBA). LBA allows the system to utilize the full capacity of a hard drive larger than 1024 cylinders (approximately 528MB). If you cannot "flash" the system's BIOS (update the BIOS with software), then you may want to consider an EIDE controller card that utilizes a "smart" BIOS. An EIDE controller card that has a "smart" BIOS will allow the system to register hard drive space above the 528MB DOS barrier. Due to the myriad of possible configurations for the large number of cards on the market, we recommend that you consult with the documentation that came with the card to configure the IDE devices in your system.

Finally, you need to make the appropriate "master" and "slave" changes to the jumper settings for your IDE devices. Most drives on the market have the jumper settings labeled on the actual drive. On some drives the jumper settings are silk-screened on the drive. You can also check the documentation that came with the drive.

What is the Basic Input Output System (BIOS)?

The Basic Input Output System (BIOS) chip on your computer controls the basic functions of your computer, including the peripherals, the ports, and the display. This BIOS chip is often referred to as the computer's nervous system for it controls the input and the responses of the computer. You can control the features of your system's BIOS through the System (CMOS) setup. Should you be asked by an AST representative to access the computer's CMOS setup, make the appropriate keystrokes when you boot the computer. We do not recommend that you make any changes to the System setup unless you are experienced with this setup utility, or you gain additional input from an AST representative. The keystrokes to access the computer's System setup vary for every AST system; however, the keystrokes are usually <CTRL+ALT+ESC>, <Delete>, or <F1>. Depending on which AST system you own, you can access the System Setup after a warm or cold boot.

You may heard of the term "flashable" BIOS. If your computer has a "flashable" BIOS, then you can update the BIOS using software. BIOS updates often make adjustments for hardware incompatibilities. You can find several BIOS updates in the Files Index of AST's Web site. To find out the latest BIOS update for your system, please check the Flash BIOS Tables. Remember, BIOS updates are usually system specific; be sure the BIOS update is correct before you begin the update.

Several new Advantage! systems included a Plug-and-Play BIOS. A PnP BIOS allows your system to automatically detect PnP devices that have been attached to the system board.

Sometimes, my mouse skips around the screen and does not move fluently. Is there a way to clean the mouse?

Cleaning the trackball will often correct common mouse problems.

Follow this procedure to clean the mouse:

  1. Remove the "O" ring that locks the mouse ball into place.
  2. Remove the ball from the socket.
  3. Use a can of compressed air, found at most computer stores, to blow air into the socket; this should remove any dust particles.
  4. Clean the roller connectors in the socket with a small amount of isopropyl alcohol on a cotton swab.

If you clean the pointing device, you will normally correct a "choppy" mouse flow. If the problem persists, you may want to re-install the mouse driver.

What should I do if the modem locks up the phone line?

Occasionally, you will notice that the modem did not disconnect properly. As a result, you continue to here the modem "squealing" when you pick up the telephone receiver. The easiest way to resolve this problem is to use a modem command to disconnect the modem. Follow this procedure to resolve this problem:

  1. Click on Start button, and then choose Programs from the menu.
  2. Choose Accessories, and then HyperTerminal from the expanded menus.
  3. Double-click on the HyperTerminal icon.
  4. Enter a name in the "Name:" field, and then click on the OK button.
  5. Enter a phone number, and then click on the OK button.
  6. Click on the OK button on the Connect screen.
  7. Enter the following command:

    8. ATH

  8. Wait for the OK response.
  9. Pick up the receiver, and check the phone line.

Why does my CD-ROM not work in DOS on my Advantage 8xx/6xx system?

Certain systems did not include Real-Mode audio drivers for the CD-ROM drive. You can download DOSBOOT.EXE from our Web site to create a bootable system disk. This disk will provide the following features:

  • Real-Mode DOS CD-ROM driver
  • Real-Mode DOS sound card driver
  • Real-Mode DOS mouse driver
  • Approximately 623KB of conventional memory

After you download DOSBOOT.EXE, follow this procedure:

  1. Right-click on the Windows® 95 Desktop. Choose New... and Folder from the menu. Label the folder DISK.
  2. Move the DOSBOOT.EXE file to the DISK folder.
  3. Double-click on the DOSBOOT.EXE icon. When you double-click on the icon, the compressed files will automatically expand in a DOS box.
  4. Insert a formatted 3.5" floppy diskette in the A: drive.
  5. Double-click on the INSTALL icon in the DISK folder.
  6. Follow the procedures in the DOS box to make the boot disk.

Once you follow this procedure, use this disk to boot the system, and then play the game or application.

I installed a retail version of Windows® 95 on my 93xx system; now my system will not recognize the CD-ROM drive. What should I do?

The retail version of Windows 95 often does not recognize the Intel 82371SB PIIX3 controller on the system board. Since the retail version of Windows 95 does not include the controller used in your system, Windows 95 will identify the controller; however, Windows 95 cannot identify the function of the controller. As a result, Windows 95 usually identifies the controller as a legacy (non-Plug-and-Play) card, and the CD-ROM drive will not function. You can obtain an update for Windows 95 from the following Web site:

http://developer.intel.com/design/motherbd/ideinfup.htm

Once you download a copy of file IDEINFUP.EXE, you can follow this procedure:  

  1. Create a new folder, and label the folder UPDATE.
  2. Save IDEINFUP.EXE to the UPDATE folder.
  3. Double-click on the IDEINFUP(1) icon. You should see a DOS box labeled IDEINFUP(1).
  4. Press <Y> to expand the compressed files. You should see the following information:

    5. Inflating: LICENSE.TXT
    Inflating: DESC.TXT
    Inflating: SW.EXE 

  5. Close the DOS box, and then return to the UPDATE folder. You should see the files that were inflated in the DOS box.
  6. Double-click on the Sw icon. You should see another DOS box.
  7. Press <Y> to expand these compressed files. You should see the following information:

    8. Inflating: SETUP.INS
    Extracting: SETUP.INI
    Inflating: SETUP.EXE
    Inflating: README.TXT
    Inflating: 371SBIDE.HLP
    Inflating: _SETUP.LIB
    Inflating: _SETUP.DLL
    Inflating: _ISDEL.EXE
    Inflating: _INST32I.EXE
    Inflating: IDEINFUP.TXT
    Inflating: DESC.TXT

     

  8. Close the DOS box and return to the UPDATE folder.
  9. Double-click on the SETUP icon. (Note: SW.EXE creates two SETUP icons; be sure to use the icon that looks like a computer next to floppy disks)
  10. Follow the step-by-step information in the Installing 82371SB IDE Controller Update Installation Wizard to complete the update.

I installed a legacy card in my Advantage! system, and now my modem does not function. What do I need to do?

Several of the new Advantage! 93xx systems shipped with 33.6Kbps Boca (Rockwell chipset) modems. We have found that when you install a legacy card in these systems, the modem will stop functioning. This issue does not occur when you install a Plug-and-Play compliant card.

If you install a legacy card in the system, then you can resolve the issue by reseating the cards in the system. Reseat the modem so that it is closest to the edge of the system. For example, install the modem in Slot 1, and then insert the legacy card in Slot 2. You can click here for an image of the riser card.
