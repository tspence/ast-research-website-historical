The hard drive field is not present in System (CMOS) Setup on my EISA-based system. How do I access this field?

When you first run the EISA Configuration Utility, several fields are extracted from System Setup that are only accessible when you run the EISACU program. If the hard drive field is not present in your system’s CMOS Setup program, you will need to run EISACU to access that field.

How do I access System (CMOS) Setup on my Premium 286 system?

AST used two different BIOS chips with Premium/286 systems. Early versions of this system shipped with a Phoenix BIOS, while the later systems shipped with an AST BIOS. Each of these different BIOS versions require different utility diskettes to access the CMOS Setup utility, "ASTSETUP." Premium/286 systems that use the Phoenix BIOS do not offer the setup routine within the ROM BIOS. You must access ASTSETUP from the utility diskette.

Change to the drive containing the diskette and enter ASTSETUP.

If your AST system has an older Phoenix BIOS, you can download DIAGS35.ARC. You will need to use the ARC600.EXE file to unzip the DIAGS35.ARC file. Both of these files can be downloaded from our BBS.

If your Premium/286 has an AST BIOS, then there are two methods to access ASTSETUP.

To access ASTSETUP from the Utility diskette, change to the drive containing the diskette and enter ASTSETUP. If you need a copy of the utility diskette, download the following file:

PR286.ZIP

You can use an alternate method to access ASTSETUP without diskette on a system that has the AST BIOS. Reboot the computer and hold down any key on the keyboard to force a keyboard error. The system will respond with a message that should prompt you to press either <F1> or <Ctrl+Alt+Esc> for Setup.

How do I access System (CMOS) Setup on my AST computer?

On most AST systems, you should see a message during a cold boot that prompts you for a series of keystrokes allowing you to access the System Setup. The majority of AST systems use the <Ctrl+Alt+Esc> keystrokes. On some AST systems, this feature is available on both warm and cold boots. The following table outlines the other exceptions to this rule:

I downloaded a BIOS update; how do I use it?

Before you install a BIOS update, you should always make sure you downloaded the appropriate file. Always consult the flash BIOS tables for the latest BIOS update. If you are unsure about a BIOS update, then contact a technical support representative for more information. Listed below is a procedure you should follow once you download a BIOS update:

Note: This procedure refers to the root directory of the hard drive as C:/> . If applicable, substitute a different drive letter.

Note: Not all BIOS updates follow this exact procedure. Always check the README.TXT file contained within the compressed BIOS file for any information about the BIOS update.

  1. Create a /TEMP directory on your hard drive.
  2. Copy the BIOS update file that you downloaded to this empty directory.
  3. Enter the following command at the DOS prompt:
    CD/TEMP
  4. Enter the name of the file that you downloaded at the C:/TEMP> prompt. When you enter this command you will extract the files contained within this compressed file.
  5. Enter "EXTRACT" or double click on the EXTRACT icon. You will see instructions to place a formatted 3.5" floppy diskette in your A: drive.
  6. Follow the on-screen procedures to extract the flash BIOS update to the diskette in drive A:.
  7. Remove the new BIOS update diskette, reboot the computer, and enter System (CMOS) Setup.
  8. Insert the BIOS diskette in the A: drive, and then select the "Update BIOS" from the Advanced Options menu.
  9. Select "Save changes, exit, and update BIOS." This option will reboot the computer and read the flash diskette in the A: drive. Following the beep codes, the BIOS will update and then the system should reboot, saving the changes.
