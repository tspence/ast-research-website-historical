How can I get more than 16 colors on my Premium II computer?

The system board in the Premium II shipped with 256KB of video RAM. This amount of video RAM is not sufficient to display Super VGA images; it will not support more than 16 colors. It is not possible to upgrade the amount of video RAM on the system board.

However, it is possible to install a third-party, ISA-based display adapter to attain the higher resolutions and greater number of colors. When you install the new card, you should disable the integrated video chipset. To disable the video chipset, move SW1-1 in the bank of eight switches to the OFF position.

If you have problems after you install the new card, the problem may be related to a conflict with the system's BIOS. Access System Setup by pressing <Ctrl+Alt+Esc> during a cold boot. In System Setup, press <PageDown> until you reach Setup Screen Two; then look for the option that reads "Shadow BIOS." If it is set to "Sys and Video," change it to read either "Sys" or "None." You may need to try both settings.

When I turn my system on there is no video and I hear two long beeps followed by three short beeps. What does this mean?

The two long beeps followed by three short beeps often indicates that the CMOS is corrupted. Often this issue occurs with systems that have a flash BIOS. To resolve the issue, reflash the system’s BIOS. If you require assistance with this procedure, or are uncertain which file to use, check the flash BIOS tables or contact a technical support representative for assistance.

How do I install/update the ATI mach64 video drivers in a DOS/Windows® 3.x environment?

The latest DOS and Windows® 3.x video drivers for the ATI mach64 video chipset are available in the Web site's Files Index. Listed below are the files that you can download:

11374317.EXE ATI mach64 v2.22 DOS/Win 3.1x/OS/2 1/2
11374318.EXE ATI mach64 v2.22 DOS/Win 3.1x/OS/2 2/2

Once you download these files, follow this procedure to create the driver diskettes and install the video drivers:

  1. Download the drivers to the hard drive. Note: This procedure refers to the root directory of the hard drive as C:\> . If applicable, substitute a different drive letter.
  2. Create two separate directories by entering the following commands at the DOS prompt:
    MD VIDEO1
    MD VIDEO2
  3. Copy each file into its own directory. Start with the first directory; copy the 11374317.EXE file to VIDEO1. Continue with the 11374318.EXE file; copying this file into VIDEO2.
  4. Enter the following at the C:\> prompt:
  5. CD VIDEO1
  6. Enter the following command at the C:\VIDEO1> prompt:
    11374317.EXE
    This command will extract the compressed files within the 11374317.EXE file.
  7. Insert a blank, formatted, 3.5" diskette into the A:\ drive.
  8. Enter the following command at the C:\VIDEO1> prompt:
    EXTRACT
    This command create the driver disk for the first video driver.
  9. Repeat this steps 4-7 to create the second driver diskette. Note: Be sure that you substitute C:\VIDEO2 for C:\VIDEO1.
  10. Once you extract the drivers to the floppy disk, read the README file on the first diskette for installation instructions. You can load the first diskette you created from file 11374317.EXE at the A:\ prompt. Enter the following command at the C:\> to load the driver update:
    A:
  11. At the A:\ prompt, enter the following command:
    INSTALL.EXE

This command will start the Windows 3.x and DOS video drivers installation. Once you install the driver update, access the Windows Setup utility in the Main directory of the Program Manager and set the video mode of choice.

How do I install the ATI mach64 video drivers in Windows® 95?

The current Windows 95 drivers for the ATI Mach64 video chipset are contained in the following files:

11374319.EXE ATI mach64 v2.22 Win95
11374321.EXE ATI mach64 v2.22 Win95
11374322.EXE ATI mach64 v2.22 Win3.1/Win95

Once you download the files follow this procedure to make the driver diskettes:

  1. Right-click on the Windows Desktop. Choose New and then Folder from the menu. Name the folder "Driver."
  2. Move file 11374319.EXE to the Driver folder.
  3. Double-click on the icon for file 11374319.EXE to expand the compressed files.
  4. Double-click on the Extract icon in the Driver folder.
  5. Follow the on-screen procedures for creating the actual driver diskette.
  6. Delete the contents of the folder.

Repeat steps 1-6 to create the disks for the remaining files.

Now you can install the new video driver:

  1. Start Windows 95.
  2. Insert the disk you created from file 11374319.EXE into the floppy drive.
  3. Click on the Start button. Click on the Run menu item. Enter the following command in the text entry area:
    A:\WIN95\SETUP.EXE
  4. Follow the instructions displayed in the ATI Setup procedures.

After you install the drivers, check the display adapter and monitor type settings in Display Properties.

What are the command line parameters for the CLMODE Utility?

Listed below are the command line parameters for the Cirrus Logic 543x CLMODE Utility Program Version 2.60:

Usage: CLMODE [[modenum][+*-]] [m[montype] | t6=x t8=x t1=x t2=x]

[modenum] mode number
[+*-] + selects 400 lines (default)
* selects 350 lines
- selects 200 lines

[montype] Monitor type
t6=x(Hz) 640x480 @ (0 = 60, 1 = 72)
t8=x(Hz) 800x600 @ (0 = 56, 1 = 60, 2 = 72)
t1=x(Hz) 1024x768 @ (0 = 87i, 1 = 60, 2 = 70, 3 = 72)
t2=x(Hz) 1280x1024 @ (0 = 87i, 1 = Not available
Example:

CLMODE 3+ (sets mode 3+)
CLMODE m2 (sets Super VGA monitor type)
CLMODE t6=0 t8=2 (sets 640x480 at 60 Hz, 800x600 at 72 Hz)

Valid monitor types:

0 VGA
1 8514
2 Super VGA
3 Extended Super VGA
4 Multi-frequency
5 Extended Multi-frequency
6 Super Multi-frequency
7 Extended Super Multi-frequency
