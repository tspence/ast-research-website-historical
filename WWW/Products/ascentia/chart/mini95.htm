 Windows 95 OSR2 (4.00.950B) - Other Information

 Item #1

Since the Mini-Dock does not have a Cardbus Controller, Cardbus Cards are not supported in the Mini-Dock.

 Item #2

Must add the following two lines to the config.sys:

Device=C:\windows\system\csmapper.sys
Device=C:\windows\system\carddrv.exe /slot=N

Note: N represents the number of slots you want to use for SRAM cards. Also, note that you can only do a full format on the SRAM card from an MS-DOS prompt.

 Item #3

Xircom Card hangs on installation (Typically, the hard drive stays active but there is no configuration). Xircom suggests on their WEB page in FAQ 3053 that in order to get many cards installed, the IrDA port should be disabled. We found it important to disable both the port in CMOS as well as the IrDA COM port in the WIN95 current configuration through Device Manager. We found that, when using the PS-CEM-28, the network side of the card will configure and works well, but the modem appears to have a problem with Windows 95s scripting as can be seen in Hyperterminal or Dial-Up-Networking. We saw the modem work only by using AT commands through a Hyperterminal window.
