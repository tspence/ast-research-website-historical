Windows NT 4.0 - Other Information

Item #1

Ositech card did not configure on IRQ 10. Would configure on IRQ 7.

Item #2

Driver does not allow you to use standard configurations. Card not supported with Card and Socket Services. No other driver was found for this Card.

Item #3

Card not recognized by System Soft after loading driver. Card will not configure.

Item #4

This card requires a registry fix for SystemSoft CardWizard (v3.00.08) before it will be recognized. The file will be named 3C575.REG and will be posted on ASTs On-Line Services

 Item #5

The Xircom CEM56-100BTX card requires version 3.10.04 of Cardwizard. Contact AST support if you are using this card.
