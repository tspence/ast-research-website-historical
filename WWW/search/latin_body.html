Reglas para especificar Búsquedas:

El buscador no distingue entre mayúsculas o minúsculas. Así, la palabra, "AST" en una pregunta emparejará con cualquiera ocurrencia en un documento como "ast". El caracter comodín y las expresiones comunes no son aceptadas como búsqueda.

Puede buscar cualquier palabra o frase, excepto las palabras que se encuentran en la lista de excepción la cual incluye palabras muy comunes como un, una, y, de, etc. las cuales son ignoradas. Puede usar Operadores lógicos o Booleanos o una combinación de palabras y frases.

Puede buscar una palabra o una frase que contenga comillas. Para buscar una palabra o una frase que contiene comillas, escriba la frase completa entre comillas y después escriba dos comillas alrededor de la palabra o palabras que desee rodear con comillas. A menudo deseará buscar sobre una familia de palabras relacionadas, por ejemplo en la palabra: junta directiva (sin doble comilla), la herramienta de búsqueda buscará las frases 'junta directiva', 'juntas directivas', etc.

Puede utilizar operadores booleanos, (en orden de prioridad) para crear combinaciones lógicas de palabras y frases:

  1. Not - El operador NOT se puede utilizar sólo tras un operador AND en consultas de contenido.
  2. And - Interpretado como el operador AND.
  3. Or - interpretado como el operador lógico OR.

Uso de la forma de búsqueda (Search Form)
Esta forma le permite controlar la manera en que la búsqueda se realiza (Search Options), así como la forma en que los resultados se despliegan al final (Display Options). Use de "Search Options" para determinar las palabras a buscar

  • Match any term (OR) Por default la búsqueda encontrará todos los documentos con cualquiera de las palabras que Ud. teclee en la caja del query.
  • Match all terms (AND) Para encontrar sólo los documentos que contengan TODAS las palabras tecleadas, utilice esta opción.


Por ejemplo, para buscar documentos que contengan las palabras 'premium' y 'server', use la expresión:

premium and server

Esta expresión buscará todos los documentos que contengan las palabras 'premium' y 'server', localizado en cualquier lugar de la página. Si la búsqueda deseada es 'premium and server', entonces teclee:

Premium "and" server

Esta expresión le mostrará todos los documentos que contienen la frase 'premium and server'. Utilice comillas dobles (") para indicar que un operador booleano se debe pasar por alto en su consulta. Por ejemplo:

Training or premium and server

Significa que el documento debe contener ambas palabras 'premium' y 'server, o únicamente contener la palabra 'training'. A la vez, la expresión:

(training or premium) and server

significa que el documento debe contener la palabra 'server', así como también cualquiera de las dos palabras 'training' o 'premium'. Se permite usar cualquier número de combinaciones de operadores booleano y paréntesis.
