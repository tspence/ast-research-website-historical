Vendor Product Description:

The APC SU1400NET provides a bidirectional communications port to coordinate a safe shutdown with most popular operating systems from Novell and Microsoft.  APC SmartUPS ships with APC's Powerchute plus power management and diagnostic software. 
Powerchute plus warns users of power problems, then automatically shuts down your system with all vital data intact, whether you are there or not.
Additionally, Powerchute plus makes use of SmartUPS' intelligent interface port to allow extensive UPS management via the LAN, SNMP or modem.

Test Results:

The APC SmartUPS SU1400NET has been tested under NT 4.0 and Netware 4.11.  NT 4.0 and Netware 4.11 were both tested on the Manhattan D & S Series.  Tests run were as follows:

1.  Simple power failures.
2.  Multiple power failures.
3.  Cold Start (start up from battery).
4.  Power failure under heavy application use. 
5.  Diagnostics.

Configuration Note:

The APC software was tested under various shutdown and power outage scenarios and presented no problems. 
the installation was straightforward and easy.  No problems were encountered with the UPS itself either.  Overall package was easy to install and use.

Installation-NT 4.0

Insert CD in drive and it will autoplay and start the the installation procedure.

Installation-IntranetWare 4.11

For Netware, be sure to be logged in to the Sys: volume before install begins.
