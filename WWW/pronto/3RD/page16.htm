Vendor Product Description:

Fast Ethernet switch that fully integrates into PC servers with 1Gbps PCI bus.  Available in 2 or 4 channels for maximum configuration flexability.  Intelligently switches network traffic between users on dedicated network segments, even if different segments are using different speeds and controls.

Test Results:

The ZNYX ZX346ARS has been tested under NT 4.0 and Netware 4.11.  NT 4.0 and Netware 4.11 were both tested on the Manhattan D & S Series.  Tests run were as follows:

1.  Transfer of 2Gb of data through a 10Base-T hub.
2.  Transfer of 2Gb of data through a 100Base-T hub.
3.  Transfer of 2Gb of data at 10Mbs and 100Mbs speeds at the same time through two different hubs to two different                                                         
    workstations. 
4.  Running MS Office 95 from multiple workstations at 10Mbs and 100Mbs.

Configuration Note:

The ZNYX card did prove to be tricky to install under NT and IntranetWare.  The user must know the device number and what PCI bus it is on to complete the installation.  ZNYX does provide a utility to do this called Zpeep located on the driver diskette but it sometimes inconsistantly reported the wrong bus number.  The user also must be aware that the firmware must be flashed every time the device driver is changed using a program called Zflash.  After resolving the previous two issues, the card performed flawlessly.

Installation-NT 4.0

1.  Install PCI card in slot.
2.  Boot up to DOS.
3.  Run Zpeep /list from the ZNYX driver diskette and make note of which PCI bus and device number the           
      card is located on.
4.  Boot up NT.
5.  Select Control Panel.
6.  Double-click on Network icon.
7.  Under the Network dialog box click the adapter tab.
8.  Click the add button.  The network adapter box will appear.
9.  Click the Have disk button.  The insert disk dialog box will appear.
10. Insert the driver diskette into drive A:
11. Press OK.
12. Driver then installs.
13. Once the driver is copied to the system a ZNYX PCIswitch setup screen will appear.  The only value that
      needs to be changed is the PCI bus number Zpeep reported the card to be on.     
14. Exit Network installation and restart the system.
15. The installation is complete.

Installation-IntranetWare 4.11

1.  Install PCI card in slot.
2.   Boot up to DOS.
3.  Run Zpeep /list from the ZNYX driver diskette and make note of which PCI bus and device number the           
      card is located on.
4.  Boot up IntranetWare.
5.  Type Load Install.
6.  Select Driver Option from the installation options menu and press Enter.
7.  Select configure Network Drivers from the additional driver actions menu.
8.  Select Discover and load additional drivers from the addtional driver actions menu.
9.    A screen will appear showing the following message: "Install found the following hardware but was un     
      able to find a matching driver: PCI LAN Controller 110D.001.0000.0000.001." Press Enter for a list of                         
      drivers to choose from or F3 to continue.
10. Then a Select Driver screeen will appear.
11. Insert the Driver diskette.
12. Press the Insert key on the keyboard and change path to A:\.
13. Press Enter and then select Rain960.lan driver.
14. After copying, a configure menu appears, and you can choose the following:
      a. Adapter instance: for more than one adapter in a machine-(don't change)
      b. PCI bus number: change to appropriate number.
      c. Node address: Don't change.
15. Select Save Parameters and load driver.
16. Then it will assign external network numbers and check for conflicts.
17. It will then present the numbers chosen and ask you to accept or change.
18. After making your choice, Select an Additional Network Driver menu will display, and ask you to select
      an additional network driver.  Select No.  Exit install and the installation is complete.
