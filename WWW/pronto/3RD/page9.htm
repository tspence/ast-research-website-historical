Vendor Product Description:

Firewall-First provides organizations with the ability to define a single, integrated security policy that spans Internet and Intranet connectivity, is transparent to network users, supports multiple protocols and networking technologies, and can be distributed across multiple platforms like NT and UNIX.

Utilizing Checkpoint's patent-pending Stateful Inspection Technology, Firewall-First builds intelligence into the network layer, creating 'virtual barriers' to control access to the network and to regulate and protect the flow of data between network resources and segments.  Because Stateful Inspection builds upon the network infrastructure itself, Firewall-First scales in tandem with the network to accommodate a growing number and type of network applications, high speed networking technologies, (e.g., 100Base-T and ATM), and business connectivity requirements.
