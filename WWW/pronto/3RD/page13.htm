Vendor Product Description:

The Deltec PRM1500a provides a bidirectional communications port to coordinate a safe shutdown with most popular operating systems from Novell and Microsoft.  Deltec PowerRite MAX UPS ship with Deltec's LANSafe III power management and diagnostic software with exclusive SafetyNet Sequential shutdown of all network devices.
LANSafe II warns users of power problems, then automatically shuts down your system with all vital data intact, whether you are there or not.  Remote control, testing and monitoring for all UPS's on the Network.  Additionally, LANSafe III makes use of PRM1500a's intelligent interface port to allow extensive UPS management via the LAN, SNMP or modem.

Test Results:

The Deltec UPS has been tested under NT 4.0 and Netware 4.11.  NT 4.0 and Netware 4.11 were both tested on the Manhattan D & S Series.  Tests run were as follows:

1.  Simple power failures.
2.  Multiple power failures.
3.  Cold Start (start up from battery).
4.  Power failure under heavy application use. 
5.  Diagnostics.

Configuration Note:

The unit performed flawlessly under various tests of power failures and other situations.  The LANSafe III software supplied presented no problems on installation and was very easy to install.  No compatability problems were observed with  the  UPS or sofware.  Overall it was simple and easy to use.

Installation-NT 4.0

Insert CD in drive and run setup from the \NTWIN\ directory to start the installation procedure.

Installation-IntranetWare 4.11

Insert CD in drive and run Setup from the \NOVWIN\ directory to start the installation procedure.
