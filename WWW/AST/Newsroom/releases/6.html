AST® COMPUTER SUPPORTS TECHNOLOGY's BEST WITH
PLANS TO INCLUDE INTEL's® LATEST 440LX AGPSET

AST Sets New Standard in Performance as it Prepares to Incorporate
Intel's® New 440LX Advanced Graphics Port into Bravo MS Commercial Desktops

IRVINE, Calif., August 25, 1997 - Continuing to surpass customers' needs for computing solutions that increase productivity, AST Computer today announced that its Bravo MS line of corporate desktop computers will include Intel's new core logic chipset, the 440LX AGPset. Intel's 440LX AGPset features a faster graphics bus, for richer and more complex graphics textures, with the ability to use system main memory when necessary, a key advantage for the high-performance user.

Other "first-time" features enabled by the 440LX AGPset on the upcoming Bravo MS Pentium® II processor-based systems include improved memory speed with SDRAM and faster IDE transfer rates with UltraDMA-33 hard disk drive. Additionally, the Bravo MS desktops will include hardware support for a power management feature called Advanced Configuration Power Interface (ACPI) which allows desktop users to have the same "suspend/resume" mode as notebook users.

"Once again AST reinforces its commitment to delivering innovative technology solutions by including the Intel 440LX AGPset in our upcoming line of Bravo MS commercial desktop computers," confirmed Laura Raybin, AST's director of Brand Marketing. "Teaming with Intel allows us to continue providing superior systems by implementing the latest in computing technology."

Advantages of AGP Technology
The AGP implementation provides a dedicated point-to-point connection between AGP and the 440LX core logic, delivering a transfer rate of up to 133 MHz. With the dedicated point-to-point AGP connection, the PCI bus is no longer burdened by graphics intensive computing. The AGP provides for scaleable graphics and will use the main memory system when necessary, providing four times the increase in bandwith over PCI-based graphics.

Additionally, DIrect Memory Execute (DIME) of textures through the AGP port reduces contention with the CPU and I/O devices for bus memory access. Intel's newest design allows graphics to utilize two buses, one that feeds instructions from the processor and another that provides graphics textures stored in main memory. The extra port to the graphics chip for memory access allows for concurrent read-in of textures from the AGP while reading and writing Z-values and pixels from local memory, in turn, allowing the CPU to write directly to system memory. With AGP, the PCI bus is used exclusively by non-graphic related peripherals such as HDD, CD-ROM, and Ethernet.
