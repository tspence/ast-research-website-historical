Vendor Product Description:

Fast Ethernet switch that fully integrates into PC or Macintosh network servers with 1 Gbps PCI bus. Available in 2 or 4 channels for maximum configuration flexibility. Intelligently switches network traffic between users on dedicated network segments even if using different network segments even if using different speeds and controls.

Test Results:

The ZYNX card has been tested under NT 4.0 and Netware 4.11. NT 4.0 and IntranetWare 4.11 were both tested on the MANHATTAN D & S Series. Tests run were as follows:

1. Transfer of 2 Gb of data through a 10BaseT hub
2. Transfer of 2 Gb of data through a 100BaseT hub
3. Transfer of 2 Gb of data at 10Mb and 100 Mb speeds at the same time through two different hubs to two different workstations.
4. Running MS Office 95 from multiple workstations at 10Mb and 100Mb speeds.

Configuration Note:

The ZYNX card did prove to be tricky to install under NT and IntranetWare. The user must know the device number and what PCI bus it is on to complete install. ZYNX does provide a utility to do this called Zpeep located on the driver diskette but it sometimes inconsistently reported the wrong bus number. The user also must be aware that the firmware must be flashed every time the device driver is changed using a program called Zflash. After resolving the previous two issues, the card performed flawlessly.

Installation-NT 4.0

1. Install card in PCI slot.
2. Boot up to DOS.
3. Run Zpeep /list from the ZYNX driver diskette and make note of which PCI bus number the card is located on.
4. Boot up to NT.
5. Select Control Panel.
6. Double-click on Network icon.
7. Under the Network dialog box click the adapter tab.
8. Click the add button. The network adapter box will appear.
9. Click the Have disk button. The insert disk dialog box will appear.
10. Insert the driver diskette into A:
11. Press OK.
12. Driver then installs.
13. Once the driver is copied to the system a ZYNX PCIswitch driver setup screen will appear. The only value that needs to be changed is the PCI bus number Zpeep reported the card to be on.
14. Exit Network installation and restart the system.
15. The installation is complete.

Installation-IntranetWare 4.11

1. Install card in PCI slot.
2. Boot up to DOS.
3. Run Zpeep /list from the ZYNX driver diskette and make note of which PCI bus and device number the card is located on.
4. Boot up IntranetWare.
5. Type Load Install.
6. Select Driver option from the installation options menu and press Enter.
7. Select Configure Network Drivers from driver options menu.
8. Select Discover and load additional drivers from the additional driver actions menu.
9. A screen will appear showing the following message: "Install found the following hardware but was unable to find a matching driver: PCI LAN Controller 110D.001.0000.0000.001." Press Enter for a list of drivers to choose from or F3 to continue.
10. Then a Select driver screen will appear.
11. Insert the Driver diskette.
12. Press the Insert key on the keyboard and change path to A:\.
13. Press Enter and then select Rain960.lan driver.
14. After copying, a configure menu appears, and you can choose the following:
15. Adapter instance: for more than one adapter in a machine-(Don't change).
16. PCI bus number: Change to appropriate number.
17. Node address: Don't change.
18. Select Save Parameters and load driver.
19. Then it will assign external network numbers and check for conflicts.
20. It will then present the numbers chosen and ask you to accept or change.
21. After making your choice, the Select an additional network driver, Menu will display and ask you to select an additional network driver. Select No. Exit install and the installation is complete.
