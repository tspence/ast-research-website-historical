Vendor Product Description:

The Deltec PRM1500a provides a bidirectional communications port to coordinate a safe shutdown with most popular operating systems from Novell and Microsoft. Deltec PowerRite Max UPS ship with Deltec's LanSafe III power management and diagnostic software with exclusive SafetyNet Sequential Shutdown of all network devices.
LanSafe III warns users of power problems, then automatically shuts down your system with all vital data intact, whether you are there or not. Remote control, testing and monitoring for all UPS's on the network.
Additionally, LANSafe III makes use of PRM1500a's intelligent interface port to allow extensive UPS management via the LAN, SNMP or modem.

Test Results:

The Deltec UPS has been tested under NT 4.0 and Netware 4.11. NT 4.0 and IntranetWare 4.11 were both tested on the Manhattan D & Manhattan S Series. Tests run were as follows:

1. Simple power failures.
2. Multiple power failures.
3. Cold Start (Start up from battery).
4. Power failure under heavy application use.
5. Diagnostics.

Configuration Note:

The unit performed flawlessly under various tests of power failures and other situations. The LANSafe III software supplied presented no problems on installation and was very easy to install. No compatibility problems were observed with the UPS or software. Overall it was simple and an easy package to use.

Installation-NT 4.0

Insert CD in drive and run Setup from the \NTWIN\ directory to start the installation procedure.

Installation-IntranetWare 4.11

Insert CD in drive and run Setup from the \NOVWIN\ directory to start the installation procedure.
