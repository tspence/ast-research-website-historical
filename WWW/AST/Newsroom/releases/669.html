Does Bravo EL offer blazingly fast graphics capabilities? Yes!

Advanced Graphics Port (AGP)

AGP is a graphics port enabled by Intel’s 440LX core logic. The AGP implementation provides a dedicated point-to-point connection between AGP and the 440LX core logic providing a transfer rate of up to 133 MHz. With a dedicated point-to-point AGP connection the PCI bus is no longer burdened by graphics. AGP provides for scaleable graphics and will use main system memory if necessary to store textures. The key benefits of AGP will be increased graphics bandwidth (at least 4x), Direct Memory Execute of textures, reduced contention with the CPU and I/O devices for bus memory access, and an extra port to the graphics chip for memory access so that it can concurrently read textures from AGP while reading/ writing Z-values and pixels from local memory, and allowing the CPU to write directly to system memory. With AGP the PCI bus is freed to use with non-graphic related peripherals such as HDD, CD-ROM , Ethernet, etc.

Is Bravo EL quieter than the competition -- making it the quietest PC in the industry? Yes!

System Noise Reduction Technology -- AST was the first computer company to bring to market business desktops that feature System Noise Reduction Technology. With this exclusive new technology, system

noise is dramatically reduced from 37 decibels to 32. Considering that a noise level of less than 25 decibels is effectively "silent" (because the human ear cannot generally discriminate between sound below this level) the Bravo EL is so quiet, it's

almost undetectable.

Today, there are approximately 30 million noise-exposed workers in the US. According to last year’s

National Institute for Occupational Safety and Health (NIOSH) technical report, a "buy-quiet" policy for

new equipment acquisitions should be adopted by management. NIOSH cites a four-part process which management can implement in order to have an effective buy-quiet policy. The process includes:

    1. Selection of specific products or operations to be targeted for noise reduction through new purchases
    2. Setting criteria for new equipment noise levels
    3. Requesting noise level specifications from manufacturers
    4. Including this noise level data in bid evaluation

NIOSH stated that 85dB is the recommended exposure limit for noise of all combined office equipment.

AST’s new Bravo MS runs at about 32dB, far exceeding the NIOSH guidelines.
