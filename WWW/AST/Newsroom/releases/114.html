Overview

AST will proudly launch the next generation of AST designed server products. The new HS Series application server will offer the highest performance server technologies, a compact full-featured chassis design, and high reliability features traditionally found only in servers positioned in the highest end of the PC server segment. AST redesigned the product from the ground up; Premium HS debuts a new electronic board-set, including a new NLX dual-Pentium II ECC 300 MHz motherboard, a new 12 drive SCSI backplane with patented AST technologies, and high-speed memory sub-system. The chassis is the smallest server chassis ever to include 12 internal hot-swap 1.6" hard drives. In addition, the chassis has been designed for ease-of-service and the flexibility to use it in either stand-alone or rack mount configurations. Redundant power supplies, RAID and fault tolerant Ethernet will be standard on all Premium HS configurations.

The Premium HS is a revolutionary server platform that brings together the best performing technologies, an easy to service and upgrade chassis design, and a full complement of fault tolerant features, into a compact, affordable and easy to use server product. The Premium HS is the perfect application server, whether it replaces an existing stand-alone server or is stacked in rack mounted server farms.

 Premium HS will replace the Manhattan S6200 server in the middle of the AST server product line. Premium HS should, however, be viewed as a revolutionary product, rather than simply an evolutionary one. Its combination of features, performance, and competitive pricing will make it one of the most attractive server offerings on the market. Additionally, early Pentium II server entries have been "processor turns" instead of revolutionary new products, which will make AST stand-out even further against its competition.

A key to the success of the product is the buy-in of sales and marketing around the world. Servers have been gaining momentum for the past nine months or so. The Premium HS product, combined with the strong commitment to market and sell the product, and the "Raving Fan" discipline, will ensure AST continues to grow as a server vendor and technology leader.

The Premium HS product will be launched in the Americas region in conjunction with the Ascentia M and the Bravo MS. The three products will show that AST is on the cutting edge of technology in all product categories. The three products will have consistent messages, focusing on innovation, life cycle management, and cost competitiveness.

Top Differentiators

Simplify Maintenance and Administration, and Reduce Upgrade Costs, and Extend Investments. The Server FlexChassis (NLX) is a new development in server design. Servers must be repaired, serviced, or upgraded as quickly as possible. The new Server FlexChassis is the easiest to service chassis ever designed for a server. The top, front and both sides are removable to allow the user full access to the interior. The motherboard slides out on rails, through the top of the system, for easy to access to memory and CPUs. Standard third-party expansion cards are also accessible through the top of the system. The power supplies and SCSI backplanes also slide in on rails.

 High Density Architecture. The FlexChassis includes a high-speed, intelligent, I2O-compliant Ethernet and SCSI RAID controller, Redundant Power Supplies, and Intel’s latest 300 MHz (SMP) processor, all featured in an extremely small platform that can be used as a stand-alone server or in a standard 19" rack cabinet. When placed in a rack, the chassis is still easy to service, because of its top-entry layout and front accessible drive bays.

 Fault Tolerance: The Premium HS brings a number of fault tolerant features down from the high-end into the middle of the product line and price range. Among these features are:

  • Redundant, Hot-Swap Power Supplies: Three power supply modules provide current to the system. If any one of the modules fails, the system can remain running on the power of the other two modules. The power supply sends an alert to the server management software and lights an LED to indicate the failure. The failed module can then be replaced without bringing the system off-line.
  • Integrated Ultra/Wide SCSI RAID: Redundant disks are now a standard in mid-range and high-end servers. But, as storage capacity needs increase, so does the demand for data integrity. AST provides the solution by featuring a three-channel RAID controller with an integrated I/O processor and 12 internal drive bays. Combined, these two features allow the user to build a high-speed, highly fault-resistant storage subsystem in any RAID configuration!
  • Integrated Intelligent Ethernet: A high-performance Ethernet solution with an integrated I/O Processor (IOP) to reduce CPU utilization. The Ethernet adapter provides fault tolerance to keep network connections active even during an Ethernet failure. (Note: this requires a second Ethernet adapter option.)
  • Server Monitoring and Embedded Diagnostics: The Premium HS has a number of server monitoring features built into the system to ensure that the server administrator is warned of a failure when, or even before, it occurs. Server monitoring measures temperature, voltage and critical data points on all boards within the system. Also, an embedded diagnostics can be run on the system to search for faults down to the component level if reoccurring trouble needs to be isolated.

High Speed Systems Architecture: AST brings a number of new standard technologies into the Premium HS. For example, AST will be the first server manufacturer to introduce an integrated II- compliant intelligent Ethernet sub-system. This will allow customers to take advantage of performance enhancing software and new peer services layer applications when they become available. As is common in most fault resistant server systems, an additional Ethernet controller can be added to further increase the availability of the entire system. Adding an additional AST intelligent controller not only increases reliability, but significantly increases LAN throughput. These new technologies make networking faster and more reliable than ever before in standard Ethernet configurations.

 In addition, the Premium HS sports a number of new high-speed technologies that are to make the server’s performance sizzle. Here are the key performance enhancing features:

  • Dual Intel Pentium II ECC 300 MHz CPUs
  • Intel 440LX core logic
  • SDRAM Memory
  • 10,000RPM Seagateâ Cheetahä Ultra/Wide SCSI disk drives
  • Integrated intelligent Ultra/Wide SCSI three-channel RAID controller with i960 CPU
  • Integrated intelligent Ethernet sub-system with i960 CPU.
  • Integrated Ultra/Wide SCSI controller for tape, CD-ROM and other peripherals
  • 24X CD-ROM for high-speed software installation

Impedance Control Technologies: SCSI backplane design is becoming more complex. As the number of drives in the system and speed of data throughput increases, the likelihood of interference increases as well. AST's impedance control technology reduces this interference and allows the backplane to run at the highest possible speeds with closely located drives. The technology is so innovative that it was awarded a patent by the United States Patent office.

New Features

Intel Pentium II Processors

Intelâ Pentium II ECC 300 MHz processor combines the power of the Pentium Proä processor with the capabilities of Intel’s MMXä media enhancement technology and double the L1 cache (32 KB), providing a high-performance solution for applications and web servers.

Based on the iCOMP 2.0 Index, the Pentium II 266 MHz and Pentium II 233 MHz processors outperform the Pentium Pro processor by 37.7% and 21.4%, respectively.

A Single Edge Contact (S.E.C.) Cartridge provides the Pentium II processor with a new innovative packaging technology that replaces Pin Grid Array (PGA) technology used on previous Intel processors. The most notable change to the Pentium II is that the level two cache and processor are integrated on a card, not on a chip, as with the Pentium Pro. This new Intel design increases the Pentium II’s manufacturability and yields, and ultimately will enable faster, more powerful Pentium II processors. SEC also allows the Premium HS to take advantage of future planned upgrades to Intel Pentium II processors -- while remaining compatible with applications running on previous members of the Intel family of microprocessors.

Intel 440LX Core Logic Chipset

A new Intel designed core logic enables the use of advanced technologies, such as SDRAM, ACPI, UltraDMA-33 and MMX. It allows better system integration between components and provides general system performance gains from previous core logic chipsets.
