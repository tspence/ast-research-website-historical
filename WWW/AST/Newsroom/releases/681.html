PREMIUM PARTNER OEM Program

Welcome to the AST Premium Partner Program!! A new Third Party Peripheral Certification program that qualifies and designates select OEM Manufacturers' hardware and software options as having interoperability with AST Servers.

This project will enable AST Sales and AST Resellers to recommend non-AST peripherals to be included as part of AST Server network solutions, and enhance AST Reseller profit opportunities.

Partners in the program have been identified, as standard add-on products used in LAN/WAN installations.

Testing is done by AST SVT (Service Verification Testing) Lab in Fort Worth, AST SCT (System Compatibility Testing) Lab in Irvine, and the AST Server Development Lab in Irvine.

The testing results are compiled by Server Product Planning and published into Compatibility Matrix's for all AST Server systems. AST Server Option Test Reports are also generated on the individual peripheral.

In addition to the documentation of testing and interoperability, the Manufacturer's part numbers are contained in the Server Matrix's, to supply to the AST Reseller Sales channel.

Worldwide Sales and Marketing contacts are also documented and published to be leveraged by AST Sales personnel and the AST Reseller Channel.
