AST® COMPUTER PROTECTS CUSTOMERS' NETWORK ASSETS
WITH CHECK POINT FIREWALL-FIRST!

AST Manhattan Internet Servers Now Come Standard With
Check Point's FireWall-First! Software At No Additional Cost

IRVINE, Calif., May 12, 1997 - Responding to businesses' growing concerns over Internet security, AST® (NASDAQ: ASTA) today announced that its servers will offer Check Point Software Technologies Ltd.'s (NASDAQ: CHKPF) Check Point FireWall-First! with no added cost to the user. Bundled on AST's Manhattan Commerce Pro and Presence Pro Internet servers, FireWall-First! is a cost-effective, pre-configured network security solution that enables businesses to set up an Internet presence easily and securely.

"AST continues to champion the small- and medium-sized business by delivering easy-to-use turnkey Internet solutions," stated Laura Raybin, director, brand marketing, AST Computer. "By offering a network security solution from the market leader without raising our server prices, AST is blazing a new trail in cost-effective server security."

FireWall-First! is a pre-configured network security solution for small- and medium-sized businesses. It employs a simple HTML-based graphical user interface that the network administrator can access from anywhere on the company's network with a Web browser. FireWall-First! supports five pre-configured Internet services including e-mail (POP3 and SMTP), Web (HTTP), file transfer (FTP), Internet Relay Chat (IRC) and Telnet.

Bundled for the first time as standard software on AST's Internet servers, FireWall-First! offers flexibility for customers to increase beyond the pre-configured number of IP addresses or upgrade to Check Point's FireWall-1 enterprise products as their network needs change.

"AST is taking aggressive measures to ensure performance, value and security for their customers," said Asheem Chandna, vice president of business development for Check Point Software Technologies, Inc. "By offering Check Point's FireWall-First! software on its Manhattan servers, AST is providing a complete Internet solution for today's businesses."

Ensuring that communication between a network and the Internet complies with a company's security policy, FireWall-First! protects users against unwanted intrusion, damage and theft. Providing the best level of security available, FireWall-First! is based on the same patented Stateful Inspection technology for controlling network traffic as Check Point's market-leading FireWall-1 enterprise security solution.

Availability

New AST Manhattan Commerce Pro and Presence Pro servers pre-loaded with Microsoft® Windows NT® Server 4.0 will offer Check Point's FireWall-First! this month.
