The AST Premium KS quad-processor SMP server builds on the advanced features and performance of the Intel® Pentium® Pro processor to deliver enterprise-level server performance to the upper end of the server market segment who demand fault tolerance and high speed performance.

AST's announcement of the GS (entry-level), HS (mid-range), and KS (high-end) servers completes the picture for offering an extremely well-rounded, high-performance server product line.

Key Sales and Marketing Messages

Scalable High-Speed Architecture

The advanced Intel architecture includes the Intel 450GX PCIset that supports up to four-way symmetric multiprocessing. The server board set has a high-throughput, dual PCI bus I/O architecture with on-board SCSI and I/O controller expansion slots, plus extensive server management and monitoring capabilities. The AST Premium KS platform packages this technology in a stand-alone cabinet with twelve "hot-swappable" SCSI hard disk bays, four peripheral bays and a 3.5" floppy disk drive. Rack mounted models of this product will be available by January 1998.

High Availability, Serviceability and Manageability

The Premium KS server contains redundant fans and power supplies for high availability. Built-in management circuitry monitors the entire system for signs of trouble and channels it to popular system management suites, like Intel LANDesk® Server Manager. The system can be configured to take immediate, automatic action in order to minimize data loss. This simplifies service and drives down support costs by allowing staff to remotely trouble-shoot and fix many problems. Security features, such as mechanical lock loop, electronic lock-out of the power, reset, and keyboard and chassis intrusion switch, help ensure high availability and security of your data.

Simplified Upgrade Paths

The base configuration of the Premium KS includes two Pentium Pro 200 MHz processors, a terminator module, and 128MB of memory. Up to 2GB of memory can be placed on the memory card to increase performance for demanding applications. An additional 4GB memory module can be added for those who just can't get enough memory.

Positioning and Target Customers

 

  • Corporate IS/IT Managers
  • High End System Integrators and Networking VARs

 

The Premium KS is targeted at VARs and end users at the higher end of the performance curve. The network managers responsible for these high-end servers are far more interested in data quality, integrity and speed than in price. They are willing to spend more for a product with a known positive reputation backed by a comprehensive service and support program.

 

 

 

Top Differentiators

 

Larger Standard Feature Set

 

To reduce VAR and end user integration time and costs, and the compatibility problems that can exist, AST includes standard a high-speed server NIC, fast/wide SCSI ports, and RAID systems. Other server manufacturers may offer theses features, but they usually come as an option.

Industry-Standard Intel-Based Architecture

The Premium KS uses field-tested server products that have been tested and certified under today's leading Network Operating Systems. End users and VARs can be confident that their applications will operate flawlessly.

Price

AST's Premium KS is typically less than that of the competition. So, in addition to offering a robust, field-tested, high-speed server platform, you get it at a lower price!

Announce Strategy

The Premium KS will be announced on November 3 along with the low-end AST Premium GS. Along with our flagship product, the Premium HS, they will complete AST's server product line. The completion of the product line with high-quality, easy-to-sell servers will simplify your efforts to satisfy the demands of the end user and VAR who require a server migration strategy and a well-round, single-vendor server solution.
