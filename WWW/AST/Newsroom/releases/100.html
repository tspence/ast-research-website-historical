AST DEPLOYS MICROSOFT WINDOWS NT SERVER AND WORKSTATION NT OPERATING SYSTEMS ON LATEST SERVERS AND DESKTOPS

Strategic Response To Corporate Needs For Robust, Powerful Commercial Systems

IRVINE, Calif., July 31, 1996 -- Arming corporate customers with the latest in technology, AST Computer (NASDAQ:ASTA) announced that its new server and commercial desktop offerings come equipped with Microsoft® Windows NT® operating systems. AST's Bravo commercial desktops are bundled with Windows NT Workstation, and AST's Manhattan™ server products come preloaded with Windows NT Server.

Future commercial desktop offerings, slated for later this summer, will be bundled with Microsoft's recently announced Windows NT Workstation 4.0 and forthcoming server products will come equipped with Windows NT Server 4.0.

"With high performance, industrial strength, reliability and security, Windows NT Workstation and Windows NT Server provide corporate users with the easiest network operating system for the most demanding business needs," stated Dan Sheppard, director of worldwide product marketing. "There has been an overwhelming demand from both our end-users and the channel for an operating system that is multi-purpose and scales from the workgroup to the enterprise -- Windows NT Server and Windows NT Workstation deliver this functionality."

Windows NT Workstation is designed to provide the highest level of system reliability and data protection for demanding environments including transaction processing, graphics handling and other mission critical applications.

With Microsoft Windows NT Server, users gain access to a robust, multi-purpose network operating system that offers high-performance file and print services while providing the infrastructure to run powerful client-server applications.

"We applaud AST for committing to deliver leading-edge technology to their customers," said Jonathan Roberts, director of desktop marketing at Microsoft Corp. "Financial applications, data analysis and a host of other data-intensive and power-hungry applications are a breeze with the combination of AST's Bravo and Manhattan products with our new Windows NT Workstation 4.0 and Windows NT Server operating systems."

Bravo Commercial Desktops
AST has designed its Bravo MS-T 6180 and 6200 desktops to take advantage of 32-bit applications and to address the needs of corporate users looking to make the transition to Windows NT Workstation and Pentium™ Pro processor technology.

The new Bravo minitowers pack a powerful punch with Intel's fastest Pentium Pro 180- and 200MHz processor engines and Intel's 440FX PCIset, letting users take advantage of today's hardware advances. Combining Intel technology with its best-of-breed Bravo PCs, AST is enabling business users to supercharge their business applications today while providing them technology for tomorrow.

"We are primed to take advantage of this powerful combination of Pentium Pro technology with Windows NT Workstation which enables us to provide our end-users and resellers with the best in cutting-edge technology -- giving them an advantage over their competition," Sheppard noted.

Manhattan Servers
The AST Manhattan server solutions are packed with the most advanced hardware and come complete with a suite of pre-installed Internet and Intranet software, designed to allow users to be up and running on the World Wide Web with minimal effort.

AST's Manhattan Commerce PRO, announced last month at PC Expo, is a turn-key Internet server that combines the best-of-breed server hardware with the most complete suite of Internet and Intranet software.

Also announced last month was the ultra-powerful Manhattan S6200, a high-performance LAN solution that delivers the latest in hardware beginning with a 200MHz Intel Pentium Pro processor for unmatched speed and performance.

"Our Manhattan servers deliver a new standard in server hardware and software configuration, representing AST's commitment to be first-to-market with the latest technology," noted Sheppard. "With the introduction of these new high-power Pentium Pro processor-based servers available with the Microsoft Windows NT Server operating system, AST is responding to the business needs of our end-users."

Pricing and Availability
Currently bundled with Windows NT operating systems, the latest Bravo and Manhattan server products will be available with Windows Server 4.0 and Windows NT Workstation 4.0 later this summer.

Pentium Pro processor-propelled Bravo PCs, with estimated street prices starting at a very aggressive $2,750, are now shipping. Estimated street prices for the Manhattan Commerce PRO begin at $8,650 and $7,500 for the Manhattan S6200. Both server products are also shipping.

Both the Bravo commercial desktops and the Manhattan server products carry a three-year on-site warranty covering hardware components. An upgraded warranty providing four-hour response time is available for Manhattan servers through the GuardianCare™ program available in the United States.
