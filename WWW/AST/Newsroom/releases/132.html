Does Bravo MS offer the worlds newest processing technology? Yes!

 IntelÒ PentiumÒ II Processors

The Intel Pentium II processor is for those who have little patience for delays. No matter what application is being run, it will run faster with the Pentium II processor. The 266 and 300 MHz Intel Pentium II processors combine the power of the Pentium Pro processor with the capabilities of Intel’s MMXä media enhancement technology and double the L1 cache (32KB), giving corporations the power they need to run 16-bit and 32-bit applications running on WindowsÒ 95 and Windows NTÒ and effectively conduct video conferencing. Software developers can use the Pentium II processors to improve full-motion video, enhance colors and achieve smooth graphics. Based on the iCOMP 2.0 Index, the 266 MHz Pentium II processor outperforms the Pentium Pro processor by 37.7%.

 Does Bravo MS offer PII and the worlds most feature-rich core logic chipset? Yes!

 Intel 440LX AGPset and Pentium II

With the Bravo MS, the Pentium II user will get full access to Intel’s most feature-rich core logic chipset, the Intel 440LX AGPset. The 440LX brings a leap forward in functionality to the Bravo MS, complementing the power of its Intel Pentium II processor. This new core logic enables the use of

advanced technology such as AGP, SDRAM, Advanced Configuration and Power Interface (ACPI), UltraDMA-33 and MMX. It allows better system integration between components and the core logic which translates into improved system performance. In addition to increased performance, the chipset enables new power management features to allow a desktop user to have the same "suspend/resume" mode as they have on their notebook.

 Does Bravo MS offer blazingly fast graphics capabilities? Yes!

 Advanced Graphics Port (AGP)

Perhaps the most exciting technology brought forth by the Pentium II members of the Bravo MS series is AGP. AGP is a new graphics port enabled by Intel’s 440LX core logic. The AGP implementation provides a dedicated point-to-point between AGP and the 440LX core logic, providing a transfer rate of up to 133 MHz. With a dedicated point-to-point AGP connection, the PCI bus is no longer burdened by graphics. AGP provides for scaleable graphics and will use main system memory, if necessary, to store textures.
